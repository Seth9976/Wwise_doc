Seeking after end of file. Playback will stop.

This can occur when using AK::SoundEngine::SeekOnEvent with a SeekPercent parameter higher than 100% or a Position parameter farther than the length of the sound file.

Recommended resolution steps:

  • Verify the parameters of the AK::SoundEngine::SeekOnEventfunction. If you have API Calls enabled in the Profiler Settings, you can find the faulty call parameters close to the error in the Capture Log.

  • Verify the length of the sound file. The error line should refer to the problematic sound, simply double-click on the "Wwise Object" column to get to the Sound.
