Import Completed

The Import Completed dialog displays the number of files that have been successfully imported, and information about the import process such as the import status or errors.

Interface element

Description

Import

A double progress bar that displays the progression of the media files that are being imported and a message that displays the number of media files that have been successfully imported.

Messages

Provides additional information about errors during the import. For example, an imported file might be missing a component.

Copies the information in the message to the Windows clipboard so that you can paste the information into a new file.

Closes the Import Completed dialog.
