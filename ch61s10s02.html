Commit Filter/Submit Changes

The Submit Changes/Commit Filter dialog displays a complete list of pending Work Units and/or source files that will be sent to the server where they will be checked-in. To give you added flexibility, you can pick and choose which files will be sent to the server.

Interface Element

Description

(Pending Files List)

A complete list of pending Work Units and/or source files that will be sent to the server where they will be checked-in.

By default all files are selected, but you can clear the selection for the files that you don't want to send to the server.

Selects all the files in the list.

Clears the selection for all files in the list.

Sends the selected pending files to the server for check-in.

Closes the dialog without checking-in the pending files.
