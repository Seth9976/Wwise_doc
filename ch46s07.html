MIDI category

The MIDI category contains MIDI properties of the selected object.

The following figures show the MIDI category for a Sound Voice.

The MIDI category for music objects allows you to specify the target and tempo of the MIDI content of an object. The target can be any playable object in the Containers hierarchy. The tempo may be specified as a global value for all MIDI content in the Interactive Music object, or it may be obtained from the MIDI files.
