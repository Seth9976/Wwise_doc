Prepared Game Syncs

The Advanced Profiler - Prepared Game Syncs tab displays information about the States or Switches that have been prepared using the PrepareGameSyncs()function. When a Game Sync is prepared, only the media files associated with that Game Sync are loaded into memory.

Interface Element

Description

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Group Name

The name of the Switch or State Group to which the prepared state or Switch belongs.

Game Sync Name

The name of the Switch or State that has been prepared using the PrepareGameSync() function.

Game Sync Type

Specifies whether the prepared Game Sync is a Switch or a State.
