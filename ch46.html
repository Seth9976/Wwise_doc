Getting to know the Property Editor

You can use the Property Editor to view and change the properties of objects. Properties are grouped into categories that each have a category filter. You can select one or more categories to display their properties in the Property Editor. Hold Ctrl or Shift to select multiple category filters. You can also select the All category to display all properties for the object. Alternatively, you can select the properties you use the most by creating a Favorites group.

The Property Editor is visible:

  • In its own window. From the menu bar, click Views > Editors > Property Editor (Ctrl+K) or click the Open in New Window icon at the bottom right of the Editor pane.

  • In the Object Tab Group of the Designer layout. See Working with Object Tabs and Object Tab Groups and Working with layouts.

Select an object in the Project Explorer to view its properties in the Property Editor. The properties and categories that are displayed depend on the type of object. See Object specific category.

Note

When you open the Property Editor in a new window, categories are added to include the properties from the primary editor. For example, for Sound SFX objects, the Effects category is displayed in the floating Property Editor.

Customizing which categories are displayed in the Property Editor

Choose which categories are displayed in the Property Editor by clicking Show/Hide Category Filters and selecting or deselecting them.

Depending on the object type, the following categories might be available:

  • General category

  • Routing category

  • Conversion category

  • Positioning category (for Containers hierarchy and most Busses hierarchy objects)

  • HDR category (for Containers hierarchy objects and Audio Busses)

  • MIDI category (for Containers hierarchy objects)

  • Advanced category (for all objects other than Auxiliary Busses)

  • Sources category

  • All category

Configuring a Favorites category

You can configure a custom category called Favorites that contains the properties and categories you want to view together. The properties you select will appear in the Favorites category under their respective categories. For example, if you configure only the Voice Pitch property as a favorite, that property appears under the General category in the Favorites list. The Favorites category is available in the tab with the star icon.

The General category in the Property Editor with only Voice Pitch selected as a Favorite.
The Favorites category selected.

To create a Favorites category:

  1. Click Show/Hide Category Filters .

  2. Do one of the following:

    • Right-click a property and select Add to Favorites. Only the property is added to the Favorites category.

    • Right-click the heading for the group of properties and select Add Group to Favorites. All the properties in that group are added to the Favorites category.

    • Right-click a property or a property group and select Configure Favorites. The Property Editor Favorites dialog opens where you can click each property to add.

Properties that are added to the Favorites category have a star beside their name.

If you have configured at least one favorite, you can filter for favorites by clicking Show/Hide Category Filters and selecting Favorites. Properties in Favorites are grouped by category. You can change the display order of the groups by clicking and dragging the title bar icon. To restore the default order, click Show/Hide Category Filters then click Reset Favorites Order to Default.

Searching by property names

Type the name of the property you are searching for in the Search field. Any properties that match what you type will be displayed in the Property Editor.

Expanding or collapsing property groups

Properties are displayed in collapsible groups. Groups can be expanded or collapsed by clicking on them or by clicking More Options and then Expand All or Collapse All. Some groups contain additional properties you can display by clicking . To have them shown always, click Show/Hide Category Filters and enable Always Show Additional Properties.

Properties displayed for Game Syncs

When a Game Sync is loaded into the Property Editor, various properties are displayed depending on the game sync type. The following table describes which properties are displayed for each Game Sync:

Game Sync

Property Editor Contents

Switch Group

Name, notes, and Switch to Game Parameter value mappings.

Switch

Name and notes.

State Group

Name, notes, and transition properties between States.

State

Name, notes, and the copying of State values.

Game Parameter

Name; Notes; Built-in Parameters to bind to; Minimum, Maximum, and Default values for the Game Parameter; and Interpolation mode values.

Trigger

Name and notes.

Properties are also displayed in the primary and secondary editors. See Primary and Secondary Editors.
