Integrity Report

The Wwise Integrity Report is where you can generate a report that contains information about your project, including errors and how to resolve them. The Integrity Report lists errors such as:

  • Missing media files

  • Missing audio or motion sources

  • Plug-in problems

  • Missing Events in SoundBanks

By double-clicking an error in the Error list, you can open a corresponding Wwise dialog where you can resolve the error, or receive further information about how to handle it. For a complete list of project errors with corresponding suggestions, refer to Integrity report issues.

You can also filter the Integrity Report to display only the types of information that you specify, such as details about the following:

  • Platforms

  • Languages

  • Audio files and sources

  • Hierarchies

  • References

  • Optimizations

Interface Element

Description

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Platforms

Specifies whether errors associated with the selected platform will be displayed in the Integrity Report or not.

Languages

Enables the language list where you can select errors related to individual languages to be displayed in the Integrity Report.

(Language List)

Specifies that the errors related to one or more selected languages will be displayed in the Integrity Report.

Reports

Audio Files and Sources

Specifies that only errors related to media files and sources will be displayed in the Integrity Report.

Hierarchies

Specifies that only errors related to the Busses and Containers hierarchies will be displayed in the Integrity Report.

Broken References

Specifies that only errors related to invalid references will be displayed in the Integrity Report.

Optimizations

Specifies that only errors related to optimizations will be displayed in the Integrity Report.

Generates the Integrity Report based on the options you have specified.

Click the Stop button to stop the generation process.

Date Generated
