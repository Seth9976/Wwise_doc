Could not allocate effect.

This error means there was not enough memory available to allocate the memory needed for the plug-in parameters. This error should always be preceded by an "Insufficient Memory" error.

Probable causes:

  • The total memory available for Wwise is too low.

  • There are too many objects loaded and created during runtime.

  • An error during custom plug-in development.

Recommended resolution steps:

  • Increase the total amount of memory available by increasing AK::MemoryArena::AkMemoryArenaSettings::uMemReservedLimit before calling AK::MemoryMgr::Init.

  • Reduce the number of objects loaded through SoundBanks.
