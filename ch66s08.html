Silence

The Silence plug-in is a sample source plug-in that you can use to create silences where no audio or motion is played.

The Silence plug-in is built into Wwise. For information on creating sources using plug-ins, see Creating sources using plug-ins.

You may notice that certain property values in both the Source and Contents Editors have an indicator beside them. This indicator shows if the property value is associated with a Game Parameter using RTPCs.

Indicator

Name

Description

RTPC - On

A property value that is tied to an in-game parameter value using RTPCs.

RTPC - Off

A property value is not tied to an in-game parameter value.
