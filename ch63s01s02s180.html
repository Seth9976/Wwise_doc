Source plug-in not found in currently loaded banks.

This error occurs if a source plug-in is used for a sound, but its properties were not included in a SoundBank, or if said SoundBank isn't loaded. The only reason why this error can occur is if some structures were explicitly excluded in the SoundBank Editor's Edit tab.

Recommended resolution step:

  • Find which SoundBank should include the Sound and Source plug-in. Open that SoundBank in the SoundBank Editor and validate that all elements related to the Sound are included.
