Profiler Statistics (Query - Dialogue Events Added)

The 'Dialogue Events Added' query shows you which Dialogue Events were triggered during the capture session. For each Dialogue Event triggered, the corresponding audio structure that was added to the dynamic sequence list, as well as how many times it was added is displayed. Several filters are also available allowing you to sort through the information quickly to determine if any actions are necessary.

Interface Element

Description

Filters

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Filter

The types of filters you can use to filter the information displayed in the Results pane. You can filter the information using any of the following filters:

  • Event Name - Displays information related to a particular Dialogue Event.

  • Target Object - Displays information related to a particular audio structure.

  • Count - Filters the information displayed in the Results pane based on the number of times a particular audio structure was added to the dynamic sequence for each Dialogue Event.

Parameters

A series of properties that allow you to define the specific criteria for each filter.

For fields that accept text strings, you can type a portion of the name; however, wildcard symbols are not supported.

Results

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Event Name

The name of the Dialogue Event that was triggered during the capture session.

Target Object

The name of the resulting audio structure that was added to the dynamic sequence list for a particular Dialogue Event.

Count

The number of times the audio structure was added to the dynamic sequence list for a particular Dialogue Event.
