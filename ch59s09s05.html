Generating SoundBanks - Stopped

The Generating SoundBanks - Stopped dialog displays the number of SoundBanks that were successfully generated before you stopped the process. It also displays a generation log that contains information about issues or potential issues that were found during the generation process. You can review these error messages within the dialog or copy them to the Windows clipboard so they can pasted into another application.

During the generation process, this dialog contains a primary progress bar that displays the overall progress of the SoundBank generation process as well as a set of secondary progress bars for each file that is being converted. The number of secondary progress bars depends on the number of processing cores on your machine. Wwise uses each processing core to convert the files in your SoundBank making the generation process as fast as possible.

Interface Element

Description

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Operation

As the SoundBanks are being generated, this column displays the name of each task that is being performed. When the generation process is interrupted, the word “Stopped” is displayed.

Progress

As the SoundBanks are being generated, this column contains a series of progress bars that display the progression of the SoundBank generation process.

A primary progress bar displays the overall progress of the SoundBank generation. A series of secondary progress bars display the progress of each operation being performed as part of the SoundBank generation. If your machine has a multi-core processor, each core is used to convert the files in your SoundBank and will have its own progress bar.

This column is empty when the generation is stopped.

Details

As the SoundBanks are being generated, this column displays additional information about the particular task being performed by each core.

This column is empty when the generation is stopped.

Results

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

SoundBanks

The names of the SoundBanks that have been generated.

Platforms

The platforms for which the SoundBanks were generated.

Language/SFX

The languages for which the SoundBanks were generated. If the SoundBanks don't contain any Voice objects, this column displays “SFX”.

Created

Displays whether the SoundBanks were successfully generated.

Status

A message that describes the status of the SoundBank generation. Any irregularities with the generation, such as missing files or SoundBanks that exceed the maximum are reported.

Log: See SoundBank Generation Tab.

Closes the SoundBanks Generation - Stopped dialog.
