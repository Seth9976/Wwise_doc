Contents Editor: Soundseed Grain plug-in

The Contents Editor gives you quick access to the most basic properties associated with the Soundseed Grain plug-in.

For a general overview of the Soundseed Grain plug-in, refer to Soundseed Grain.

Interface Element

Description

Name

The name of the Soundseed Grain plug-in.

To the right of the plug-in name is, if selected, the associated audio source filename.
