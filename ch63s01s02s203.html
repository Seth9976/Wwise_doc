Unexpected call to PrepareGameSyncs.

AK::SoundEngine::PrepareGameSyncs can only be called successfully if AkInitSettings::bEnableGameSyncPreparation is set to true. The call is ignored. See AK::SoundEngine::PrepareGameSyncs.
