Contents Editor: Audio Input

The Contents Editor gives you quick access to the properties associated with the Audio Input. You must click the Audio Input source to display the associated column headings.

Interface Element

Description

Click the Configure Columns shortcut (right-click) option from the column header band.

The Object Property Settings opens. Specify which columns to display and their order.

Note

You cannot configure the columns for Source Contents Editors.

Name

The name of the audio input source plug-in.

Use

Determines which source within the sound object is:

  • Played during playback.

  • Included in the SoundBanks.

  • This option is only visible when you have more than one source within the sound object.

Gain

The level or amplitude of the audio output.

Default value: -12

Range: -96.3 to 0

Units: dB

Notes

Any additional information about the audio input source.

Adds a new source to the sound object.

When you click the Add Source button a list of possible sources that can be added is displayed.
