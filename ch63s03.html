Advanced Profiler

The Advanced Profiler displays a wide range of very detailed information about the inner workings of the sound engine. For example, you can get information about the memory categories registered in the sound engine's Memory Manager. You can view information as it is being captured by clicking the Show Live Data button, , or you can view the information at a specific point in time by dragging the Performance Monitor time cursor in the graph view. You can also press Alt+Comma (,) to move one audio frame to the left and Alt+Period (.) to move one audio frame to the right.

The Advanced Profiler is divided into the following tabs:

  • Voices Graph

  • Voices

  • Busses

  • Sends

  • Sidechain Mixes

  • Audio Devices

  • Emitter-Listener Associations

  • Memory

  • Memory Arenas

  • Streams

  • Streaming Devices

  • CPU

  • Obs/Occ

  • Listeners

  • SoundBanks

  • Loaded Media

  • Prepared Events and Busses

  • Prepared Game Syncs

  • API Calls
