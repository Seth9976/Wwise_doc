Contents Editor: Soundseed Wind

The Contents Editor gives you quick access to some of the most common properties associated with the Soundseed Wind plug-in. You must click the Soundseed Wind source to display the associated column headings.

For a general overview of the Soundseed Wind plug-in, refer to Soundseed Air Wind.

Interface Element

Description

Name

The name of the Soundseed Wind plug-in.

Use

Determines which source within the sound object is:

  • Played during play back.

  • Included in SoundBanks.

  • This option is only visible when you have more than one source within the sound object.

Notes

Any additional information about the Soundseed Wind plug-in.

Wind Speed

The average velocity of the wind. Wind speed is interpreted in terms of an offset in pitch applied to the deflectors in a scene.

Default value: 0

Range: -2400 to 2400

Direction

The path or course in which the wind is moving. A value of 0 degrees means that the wind is coming from the front or north and a value of 180 or -180 means that the wind is coming from the back or south.

Default value: 0
Slider Range: -180 to 180
Units: Degrees

Variability

The amount by which the wind speed can change over time due to wind gusts.

Default value: 0.25
Slider Range: 0 to 1

Gustiness

The likelihood that the wind acceleration will change over time.

Default value: 0.5
Slider Range: 0 to 1

Gain Offset

A global property that defines the amount by which all deflectors’ gain is offset. This property can be used as a master gain control.

Default value: 0.0

Range: -96.3 to 24.0

Units: dB

Adds a new source to the sound object.

When you click the Add Source button a list of possible Source plug-ins that can be added is displayed.
