Source Editor: Audio Input

The Source Editor displays all the basic properties associated with the selected instance of the Audio Input plug-in. You can set its Gain or apply an RTPC to it.

For information on integrating the Audio Input, refer to the Wwise SDK documentation.

Interface Element

Description

Name

The name given to this instance of the Audio Input.

Source Plug-in

The type of source plug-in.

Notes

Any additional information about the Audio Input.

Effect Settings

Audio Input

Gain

The level or amplitude of the audio output.

Default value: -12

Range: -96.3 to 0

Units: dB

RTPC

The RTPC tab allows for creation of RTPCs on the Audio Input's Gain value. Refer to the RTPC tab page for general information.
