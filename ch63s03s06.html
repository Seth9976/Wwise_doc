Audio Devices

The Advanced Profiler - Audio Devices tab displays information about each of the Audio Devices active in the sound engine. Active devices are either specified at initialization of the sound engine using AkOutputSettings when calling AK::SoundEngine::Init, or they are added by calling AK::SoundEngine::AddOutput, replaced by calling AK::SoundEngine::ReplaceOutput, or removed by calling AK::SoundEngine::RemoveOutput.

Interface Element

Description

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Audio Devices

Identifies the Audio Device ShareSet from the project.

Device ID

Identifies the specific device using this Audio Device ShareSet.

Specified Listeners

Shows listeners provided to AK::SoundEngine::AddOutput. Specified Listeners are only taken into account when multiple Audio Devices using the same ShareSet are active simultaneously. When using multiple Audio Devices sharing the same ShareSet, Specified Listeners identifies which listeners should output to this device. Audio Devices with no Specified Listeners will accept sounds from listeners not specified by any other Audio Devices using the same ShareSet.
