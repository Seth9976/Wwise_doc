Contents Editor: Motion

The Contents Editor is very basic for the Motion plug-in. Only the standard properties are accessible in the Contents Editor.

Interface Element

Description

Click the Configure Columns shortcut (right-click) option from the column header band.

The Object Property Settings opens. Specify which columns to display and their order.

Note

You cannot configure the columns for Source Contents Editors.

Name

The name of the Motion plug-in.

Use

Determines which source within the motion object is:

  • Played during playback

  • Included in the SoundBanks.

This option is only visible when you have more than one source within the motion object.

Notes

Any additional information about the Motion source.

Adds a new source to the sound object.

When you click the Add Source button a list of possible sources that can be added is displayed.
