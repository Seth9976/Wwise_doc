Insufficient memory preparing media. Sound will not play.

There isn't enough memory available to load the media related to an Event or a Switch.

Recommended resolution steps:

  • Split SoundBanks into smaller SoundBanks, loaded dynamically at appropriate times.

  • Unload unused SoundBanks.

  • Increase the total amount of memory available by increasing AK::MemoryArena::AkMemoryArenaSettings::uMemReservedLimit before calling AK::MemoryMgr::Init.
