Transitions tab

The main part of the Transitions tab is the transition matrix, a list of rules that defines how each object within a Music Switch or Playlist Container transitions to every other object within the container. It also contains controls for customizing the properties of each source and destination in a transition, and for specifying the use and properties of transition segments.

See Working with transitions.
