Work Units

A project can be organized within the Project Explorer using Work Units. These can be created as elements within any tab of the Project Explorer, and they allow your project to be parceled into distinct pieces that can more easily be managed within a source control system (as might be set up in the Source Control group of the Projects Settings General tab of the Project Settings dialog).

Adding a Work Unit to your Project Explorer hierarchy prompts the New Work Unit dialog, where you can name the element and its corresponding WWU file.
