Views

The many Wwise views are opened in contextually appropriate places throughout the authoring tool, such as within the Project Explorer or via its Shortcut Menu Search Options. Some views with a broader project scope are, however, limited to being opened from the Views menu, through Customizing keyboard shortcuts and commands, or by way of loading specific layouts. These views include the following:

Project Level Validation

  • Integrity Report

Detail and Playback for Audio Structures

  • Details

  • Transport Control

  • Audio Device Meter

  • Loudness Meter

  • Meter

Layout-specific

  • SoundBank Manager

  • Busses Console

Tip

If a view name is unavailable in the Views menu, then the view is already open. If you cannot see it, it might be minimized or otherwise hidden from view. To find it, try to activate it from the Windows dialog.
