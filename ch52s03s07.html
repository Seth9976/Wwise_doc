Metadata tab

The Metadata tab allows you to associate an unlimited number of Metadata plug-ins with an object. When you associate a Metadata plug-in with an object, you must choose the Metadata plug-in type, and whether the Metadata plug-in is a ShareSet or a custom instance. Clicking the Edit […] button opens the Metadata Editor, allowing you to edit all properties of the Metadata plug-in.
