Using the Control Surface Toolbar

The Control Surface Toolbar appears near the top of the Wwise interface when a Control Surface Session is active. You can open and close the toolbar by clicking the activation icon in the Control Surface Bindings view or in a Control Surface Session Object Tab.

Control Surface Toolbar
Activate/Deactivate Control Surface Session

The Control Surface Toolbar contains the following elements, arranged from left to right:

Interface element

Description

Active Control Surface SessionShows the currently active Control Surface Session. Double-click to open the session in an Object Tab.
Opens a list of Control Surface Sessions. Switch to a different session by selecting one from the list.

Current Selection Object

Shows the currently selected object.

Last Property Changed

Shows the last object, property, and value (as a slider) modified with a control surface's binding.

Loaded Binding Groups

These buttons show which groups, from among the following, are active and inactive:

  • Global: The target object is defined in binding.

  • Current Selection: The target object is the last selected one.

  • View Group: The target object is loaded in the specified group via the specified view. For example, the button might be written YourGroup:Soundcaster, which indicates that your project has a defined View Group called "YourGroup" that is used within the Soundcaster. There can be multiple View Groups defined in different views, so more than one button might be visible.

Click a group name to activate or deactivate the group. Colors change to indicate the group status:

  • Active: orange

  • Inactive: gray

Deactivates the Control Surface Session and closes the Control Surface Toolbar.
