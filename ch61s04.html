Game Syncs Tab

The Game Syncs tab of the Project Explorer displays a complete list of Game Syncs - the States, Switches, Triggers, and Game Parameters in your project organized into Work Units and folders. You can create and manage your Game Syncs and the individual Game Sync Work Units in this tab as well as navigate through the Game Sync Work Units by clicking the plus (+) and minus (-) signs to expand and collapse the Work Unit folders. The Game Syncs tab also includes a toolbar that allows you to quickly add game syncs, Work Units, and folders to your project hierarchy.

Interface Element

Description

(Toolbar)

Displays one or more icons representing the project elements that can be added as a parent or child of the selected node in the project explorer.

By default, the toolbar displays all project elements that can be added as a child of the selected node. To display the possible parent project elements, press the Shift key.

To add a child to the selected node, simply click one of the active icons in the icon bar. To add a parent to the selected node, simply Shift+click one of the active icons in the icon bar.

Different icons will become active depending on the type of object you currently have selected in the hierarchy.

Switches

Displays the complete list of Switch Groups and Switches sorted by Virtual Folders and Switch Work Units for your project. Double-click a Switch Group or Switch to open the corresponding property editor.

States

Displays the complete list of State Groups and States sorted by Virtual Folders and State Work Units for your project. Double-click a State or State Group to open the corresponding property editor.

Game Parameters

Displays the complete list of Game Parameters sorted by Virtual Folders and Work Units for your project. Double-click a Game Parameter to open the Game Parameter Property Editor.

Triggers

Displays the complete list of Triggers sorted by Virtual Folders and Work Units for your project. Double-click a Trigger to open the Trigger Property Editor.
