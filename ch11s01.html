The importing process

Before importing your files into your Wwise project, we recommend you first look at what kinds of files Wwise supports, and where you want to store these media files. Also see Understanding file length limitations and naming conventions.

Note

In addition to the PCM audio file formats, it's also possible to import MIDI files. Refer to Importing MIDI files.

What happens during the importing process?

Wwise performs several activities during the import process. The conversion of media files is done separately. For more information on converting media files for each platform, refer to Managing audio file conversion.

The import process includes the following operations:

  • Wwise validates the original media files before copying them into the project's Originals folder.

  • Audio sources are created for the media files.

  • Sound or music objects that contain audio sources are created in Wwise and are displayed in their respective hierarchies in the Project Explorer's Audio tab.
