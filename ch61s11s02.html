Save Preset dialog

The Save Preset dialog allows you to save Presets for any of the following:

  • Property values and settings for objects within the Busses and Containers hierarchies

  • Positioning values and settings

  • Attenuation values and settings

  • Effect values in the Effect Editor

  • Metadata values in the Metadata Editor

When you save a Preset, it saves every value on every tab within a view.

Interface Element

Description

Name

The name you want to give to the current Preset.

Notes

Any additional information you want to add that explains the current Preset.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Name

The names of Presets that have already been saved.

Notes

Additional information about existing Presets.

Deletes the Preset that is currently selected in the list.

Saves the entire view's current settings as a Preset.

Closes the dialog without saving a Preset.
