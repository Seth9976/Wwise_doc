AK::SoundEngine::SetMultiplePositions: Too many positions.

The in_pPositions parameter array of AK::SoundEngine::SetMultiplePositions is too large to fit in the command queue.

Recommended resolution steps:

  • Reduce the number of positions used in this function.

  • Enlarge the command queue memory by changing AkInitSettings.uCommandQueueSize before AK::SoundEngine::Init. See Advanced Sound Engine Integration for more information on initialization settings.
