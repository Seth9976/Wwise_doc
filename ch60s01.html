Edit

The Edit Menu provides some convenient, common options for working with Wwise:

  • Undo: Reverts the last Wwise property change. A short description of that change appears after Undo. (Ctrl+Z)

  • Redo: Applies the last reverted change. A short description of that change appears after Redo. (Ctrl+Y)

  • Cut: Moves selected editable text, such as in a Note, to the clipboard. (Ctrl+X)

  • Copy: Adds selected editable text, such as in a Note, to the clipboard. (Ctrl+Y)

  • Paste: Applies the clipboard's content to the selected location. (Ctrl+C)

  • Convert: Opens the Audio file conversion dialog dialog for conversion of the currently selected audio structure. (Shift+C)

  • Navigate Backward: Loads the last closed selection. (Backspace, Alt+Left)

  • Navigate Forward: Loads the next selection, which is possible only if you have navigated backward. (Shift+Backspace, Alt+Right)

  • Navigate Up: Loads the parent object of the current selection. (Alt+up)
