Platform Support

This section lists the supported platforms, architectures, and SDK versions for the speechEngine plug-in.

Windows:

  • Architectures:

    • x64

  • SDK versions:

    • vc160

    • vc170

    • vc180

Android:

  • Architectures:

    • arm64-v8a

    • x86_x64

  • SDK versions:

    • Android NDK r25b

    • Minimum API 21

iOS:

  • Architectures:

    • iphoneos (simulator not supported)

  • SDK versions:

    • Xcode 16

    • Xcode 26

Linux:

  • Architectures:

    • x64
