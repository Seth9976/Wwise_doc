Source Editor: Silence

The Source Editor displays all the properties associated with the Silence plug-in. You can add silences of different lengths by modifying the property values.

Interface Element

Description

Name

The user-defined name given to this instance of the Silence plug-in.

Source Plug-in

The type of source plug-in.

Notes

Any additional information about the Silence plug-in.

Length

Duration

The duration of the silence source.

Default value: 1.0

Range: 0.001 to 3600

Units: s

Random Min

An offset from the Duration value that defines the minimum possible length of the silence.

Since you can’t apply a randomizer to the silence plug-in, the Rand Min and Rand Max properties act as randomizers in specifying the minimum and maximum offset values.

Default value: 0.0

Range: -3600 to 0

Units: s

Random Max

An offset from the Duration value that defines the maximum possible length of the silence.

Since you can’t apply a randomizer to the silence plug-in, the Rand Min and Rand Max properties act as randomizers in specifying the minimum and maximum offset values.

Default value: 0.0

Range: 0 to 3600

Units: s
