Containers hierarchy: sound and motion objects

The Containers hierarchy is a hierarchical series of sound, motion, and music objects, all of which can be organized within a variety of different containers. Descriptions of each of the sound and motion object property options can be found in the following Property Editor descriptions. For information on music objects, see Containers hierarchy: music objects.

  • Property Editor: Property Container

  • Property Editor: Random Container and Sequence Container

  • Property Editor: Switch Container

  • Property Editor: Blend Container

  • Property Editor: Sound SFX and Sound Voice

In addition to the General Settings described in the pages above, each object has a number of other Property Editor tabs described here:

  • Common tabs and categories: Containers hierarchy objects

When a Containers hierarchy object is selected in the Project Explorer, the corresponding Contents Editor loads some information about its child or source objects.
