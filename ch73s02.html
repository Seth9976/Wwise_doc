Generating Packages dialog

The Generating Packages dialog displays information about each file that was added to the package during the generation process. Any issues found during the generation process will be highlighted in yellow.

Interface Element

Description

(Type - Color)

Displays the color associated with the different types of messages. The following types exist in the SoundBank Log:

White - For general messages.

Yellow - For package generation warnings.

Red - For package generation errors.

(Message)

The location of the original file that was added to the package. If an issue is encountered during generation, information about the issue will be displayed beside a yellow circle.

Copies the information in the log to the Windows clipboard so that you can paste the information into a new file.

Stops the package generation process.

Closes the Building Packages dialog.
