Timeline Configuration

The Timeline Configuration dialog is where you define the properties and behaviors of the Position Editor timeline. You can define the length of the timeline, the time intervals between new control point keys, and how existing keyframes will react when the timeline is shortened or lengthened.

Interface Element

Description

Length

Defines the length of the timeline in mm:ss.ms. The maximum length of the timeline is 59:59:999.

Stretch proportionally

When changes are made to the length of the timeline, the existing control point keys are re-positioned to maintain their relative positions to one another on the timeline.

This option is only available when the timeline is in Non-Linear mode.

Preserve key values

Control point keys on the timeline keep their respective positions as changes are made to the length of the timeline.

Control point keys that exceed the length of the new timeline will be deleted.

This option is only available when the timeline is in Non-Linear mode.

Insert Key Every

Determines the amount of time between control point keys when one is inserted on the timeline. This option only applies in non-linear mode.

The default value is one second (00:01:000).

Closes the Time Configuration dialog and applies the settings you selected.

Closes the Time Configuration dialog without applying any changes to the timeline.
