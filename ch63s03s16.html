Loaded Media

The Advanced Profiler - Loaded Media tab displays information about the media files that have been loaded into memory as a result of the PrepareEvent()and PrepareGameSyncs() functions.

Interface Element

Description

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Media File Name

The name of the media file loaded into memory.

Size

The size of the media file loaded into memory.

Format

The audio format of the media file loaded into memory.

Note

An Opus or a Vorbis media file that has already been decoded by a call to DecodeBank() will appear as PCM.

Tip

Refer to About audio formats to learn more about the formats supported by Wwise.

SoundBank Name

The name of the SoundBank that contains the media file.
