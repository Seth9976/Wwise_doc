Mixing Desk Settings

The Mixing Desk Settings dialog allows you to enable or disable the type of information displayed in the Mixing Desk. You can also define the height of the vertical faders and the meters.

Interface Element

Description

Monitoring

Activity

Determines whether the monitoring information will be displayed in the Mixing Desk. The following activities can be monitored in the Mixing Desk:

  • Voice playback

  • Bus ducking

  • Effect bypass

Mute/Solo

Determines whether the mute or solo functionality will be available in the Mixing Desk.

Meter

Determines whether the meter will be available in the Mixing Desk.

Bus

Output Bus

Determines whether the name of the Bus through which an object is output will be displayed in the Mixing Desk.

Is Ducking

Determines whether the name of the Bus or Busses that are ducked when audio is passed through the current Bus will be displayed in the Mixing Desk.

Output Bus Volume
(horizontal slider)

Determines whether a volume slider will be displayed in the mixer strip. This slider allows you to control the volume levels of an object's specified Output Bus.

Output Bus Low-pass Filter
(horizontal slider)

Determines whether a Low-pass filter slider will be displayed in the mixer strip. This slider allows you to control the level at which the filter is applied on the object's Output Bus.

Output Bus High-pass Filter
(horizontal slider)

Determines whether a High-pass filter slider will be displayed in the mixer strip. This slider allows you to control the level at which the filter is applied on the object's Output Bus.

Game-Defined Auxiliary Sends Volume
(horizontal slider)

Determines whether the Auxiliary Send Bus at the specified index will be displayed in Mixing Desk.

User-Defined Auxiliary Send 0-3

Determines whether the Auxiliary Send Bus at the specified index will be displayed in Mixing Desk.

User-Defined Auxiliary Send Volume 0-3

Determines whether the Auxiliary Send volume at the specified index will be displayed in Mixing Desk.

Positioning

Attenuation

Determines whether Attenuation ShareSet information will be displayed in the Mixing Desk.

Center %

Determines whether center % information will be displayed in the Mixing Desk.

Effects

Effect 0-3

Determines whether Effect information related to Effect slot 0-3 will be displayed in the Mixing Desk.

States

State Group

Determines whether the State Group information for each object will be displayed in the Mixing Desk.

State Properties

Determines whether the State Properties added to each object will be displayed in the Mixing Desk.

Properties

RTPC

Determines whether an RTPC control will be displayed in the Mixing Desk.

Bus Volume
(horizontal slider)

Determines whether Bus volume information will be displayed in the Mixing Desk.

Voice Volume
(horizontal slider)

Determines whether a horizontal volume slider will be displayed in the mixer strip. This slider allows you to control the volume levels of each object or bus.

Voice Pitch
(horizontal slider)

Determines whether a horizontal Pitch slider will be displayed in the mixer strip. This slider allows you to control the pitch values for each object or bus.

Voice Low-pass filter
(horizontal slider)

Determines whether a horizontal Low-pass filter slider will be displayed in the mixer strip. This slider allows you to control the LPF values (for more detail, see Wwise LPF and HPF value cutoff frequencies) for each object or bus.

Voice High-pass filter
(horizontal slider)

Determines whether a horizontal High-pass filter slider will be displayed in the mixer strip. This slider allows you to control the HPF values for each object or bus.

Make-up Gain
(vertical fader)

Determines whether a vertical Make-up Gain fader for the object will be displayed in the Mixing Desk.

Bus Volume
(vertical fader)

Determines whether a vertical volume fader will be displayed in the mixer strip. This fader allows you to control the volume levels of each Bus.

Voice Pitch
(vertical fader)

Determines whether a vertical pitch fader will be displayed in the mixer strip. This fader allows you to control the pitch values for each object or bus.

Voice Low-pass filter
(vertical fader)

Determines whether a vertical low-pass filter fader will be displayed in the mixer strip. This fader allows you to control the LPF values (for more detail, see Wwise LPF and HPF value cutoff frequencies) for each object or bus.

Voice High-pass filter
(vertical fader)

Determines whether a vertical high-pass filter fader will be displayed in the mixer strip. This fader allows you to control the HPF values for each object or bus.

Make-up Gain
(vertical fader)

Determines whether the Make-up Gain information will be displayed in the Mixing Desk.

Other

Object Name

Determines whether the name of the object or bus will be displayed at the bottom of each mixer strip.

Vertical fader height

Determines the height, in pixels, of the vertical faders in the Mixing Desk.

Meter height

Determines the height, in pixels, of the meters in the Mixing Desk.

Applies the changes you made to the Mixing Desk settings.

Closes the Mixing Desk Settings dialog without applying your settings.
