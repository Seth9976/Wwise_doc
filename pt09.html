Audiokinetic plug-ins

This section provides detailed descriptions of the various plug-ins for Wwise developed by Audiokinetic.

To view all available Audiokinetic plug-ins, open the Plug-ins & Assets page and click Filters > Audiokinetic.

Some plug-ins are built-in to Wwise and others must be installed through the Launcher. For information on installing plug-ins, see Installing Plug-ins in the Audiokinetic Launcher guide. You can install plug-ins using the Launcher and then use them in your project in Wwise. However, if you don't apply a license to your project that covers these plug-ins, you get an error when generating SoundBanks. For more information, see Projects in the Audiokinetic Launcher guide.

The following types of plug-ins are available:

  • Audio Device plug-ins—descriptive and reference material on plug-ins that function as an Audio Device: third-party and custom Audio Devices and AK ASIO Output.

  • Audio Device Effect plug-ins—descriptive and reference material on plug-ins inserted as Effects on Audio Devices.

  • Source plug-ins—descriptive and reference material on plug-ins used to generate audio or motion in place of a source file.

  • Effect plug-ins—descriptive and reference material on plug-ins inserted as Effects used to enhance sounds, motion, or music.

  • VST3 plug-ins—installation instructions as well as descriptive and reference material on VST plug-ins.
