Navigating Control Surface Sessions

You can open a single Control Surface Session in the Control Surface Bindings view. However, if you need to manage additional sessions, you can open them in Object Tabs. Any sessions that you open from the Project Explorer appear in Object Tabs, and are inactive by default. Any session that you open in the Bindings view is active by default.

To change the active session, click the activation icon () in the view or Object Tab of the desired session. An orange icon indicates that the session is active, and a gray icon indicates that it's inactive. The active session also appears in the Control Surface Toolbar.
