Audio Object Metadata

The Audio Object Metadata view provides detailed information about the metadata of the currently selected Audio Object. In order to have content shown in this view, you must select an Audio Object in the Audio Object List. The information shown in this view is based on the current profiler cursor time.

Metadata is a collection of information stored along with the Audio Object. Metadata includes the positioning information from Wwise, as well as the custom Metadata plug-in’s information inserted from the Containers hierarchy or the Busses Hierarchy.

Metadata can be used by an Effect plug-in or by an endpoint to render Audio Objects. For more information on Metadata, refer to Metadata.

Note

To have information in this view, you must enable Audio Object Data in the Profiler Settings dialog prior to capturing data, and you must select a point in time from the capture.

Interface Element

Description

Click the View Settings icon in the upper right corner of the view to open the Profiler Settings dialog dialog, where you can specify the type of data to capture.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Metadata

Shows the name of the metadata item and the names of individual values in the metadata.

Value

Shows the value for a metadata item.

Source

Shows where this metadata was set in the Wwise project, if it is a user-applied metadata. For example, if the metadata was added to a Property Container object, you would see the Property Container object here. Double-clicking the object will open the view where the metadata is defined.

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.
