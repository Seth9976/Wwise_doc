Too many event posts on event.

There are too many AkMIDIPost items in the parameters of the function AK::SoundEngine::PostMIDIOnEvent; the whole array doesn't fit in the command queue.

Recommended resolution steps:

  • Reduce the number of AkMIDIPost items used in the function.

  • Enlarge the command queue memory by changing AkInitSettings.uCommandQueueSize before AK::SoundEngine::Init. See Advanced Sound Engine Integration for more information on initialization settings.
