FLT_MAX not supported in function ...

The maximum IEEE float value FLT_MAX can not be used as an input parameter value in some functions because it leads to undefined behaviors. The function call will be ignored.

Recommended resolution step:

  • Look for the reported function usage in your game code and validate that the values used are valid float values.
