State Properties Dialog

The State Properties dialog allows you to add and remove State properties from the object. Unlike the Configure Columns dialog, this does not just remove from view the property column in the table. Additionally, the column order is fixed in the States tab; you cannot rearrange them.

Interface Element

Description

Properties

A list of all the State properties applicable to the selected object.

Enable the associated check box to apply a property to the object.

Selects all the properties in the list.

Clears the selection for all properties in the list.

Closes the State Properties dialog and applies any changes to the selected object's States tab table.

Closes the State Properties dialog without changing the selected object's States tab table.
