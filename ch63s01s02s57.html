Dynamic Sequence already closed.

An operation on a Dynamic Sequence was attempted on a closed sequence. Make sure that all Dynamic Sequence function calls occur before AK::SoundEngine::DynamicSequence::Closeis called. You can validate the order of calls as they were received by Wwise by enabling API Calls in the Profiler Settings.
