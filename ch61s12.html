Searches and Project-wide Editing

In addition to the creation of project-wide queries from within the Queries Tab, there are five different views for searching and editing your project, all of which are accessible through the shortcut menu of most Project Explorer elements.


Shortcut Menu Search Options

  • Batch Rename: Opens the selected elements (there must be at least two) in the Batch Rename View.

  • Show in List View: Opens the selected element in the List View.

  • Show in Multi Editor: Opens the selected element in the Multi Editor.

  • Show in Schematic View: Opens the selected element in the Schematic View.

  • Find all references: Opens the selected element in the Reference View.

Tip

The columns of most of the search and editing views, like the Query Editor, can be adjusted within the Object Property Settings dialog.
