File or loop region is too small to be played properly.

The loop points in the file are not far enough apart. If there are no loop points in the file, then the file is set to loop in its entirety but is too short.

Probable cause:

  • This error may be reported by hardware decoders when the looping region is shorter than one audio frame.

Note

The length of the audio frame is defined at the sound engine's initialization by the value of AkInitSettings.uNumSamplesPerFrame.

Recommended resolution steps:

  • Enlarge the looping region in the Source Editor in Wwise. Then regenerate and redeploy your SoundBanks.

  • Use a different codec. All software codecs can handle tiny loops. Then regenerate and redeploy your SoundBanks.
