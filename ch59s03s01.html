Logs Settings

The Logs Settings dialog allows you to filter out messages, warnings, and errors displayed in three of the five tabs of the Logs view, namely the General Tab, Opening and closing projects, and WAAPI Tab. For the other two, the Conversion Tab and the SoundBank Generation Tab, go to the Logs tab of the Project Settings dialog of the Project Settings dialog.

Interface Element

Description

Ignore

Determines whether the corresponding message type will be displayed the next time Wwise outputs the specified log. By default, all message types will be displayed. When enabling this check box, the message type will be ignored by Wwise and will not be displayed in the log.

Message

Describes the issue encountered.

Tip

You can also add messages to the Log Ignore List by right-clicking one or more messages directly in the log and selecting Add Message(s) to Ignore List.
