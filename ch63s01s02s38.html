Cannot open file ... in path(s): ...

The file specified was found but is not accessible in any of the listed paths. This is usually a file permission error, but different OS might give other errors that lead to this message.

If the file was not found, the error File ... not found in path(s): ... would be printed instead.

Probable causes:

  • The file permissions prevent the Read operation by the current user.

  • Unknown error returned by the operating system.

Recommended resolution step:

  • Check the file permissions in the deployment folder of the game for the specified file.
