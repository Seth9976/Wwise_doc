Reset Playlist action with a game object specific scope is ignored on a global container.

Sequence or Random Container's playlist are always specific to a game object. Therefore posting an Event that contains a Reset Playlist action on a Container that has the Scope property set to Global has no effect.

Recommended resolution steps:

  • Remove the Reset Playlist action from the Event.

  • Change the Container Scope property to Game Object.
