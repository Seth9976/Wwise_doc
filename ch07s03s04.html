Managing messages that appear in the logs

There might be situations where you no longer want or need specific warnings and/or messages to be displayed in the SoundBank or Conversion logs during the generation process. To prevent these types of warnings and messages from appearing in the SoundBank and Conversion logs, you can add them to the Log Ignore list in the Logs tab of the Project Settings dialog. When you want to include these messages again, you can remove the message types from the Log Ignore List. This list of messages is managed at the project level in the Project Settings dialog.

To add messages to the Log Ignore List:

  1. From the menu bar, click Project > Project Settings.

  2. In the Project Settings dialog, select the Logs tab.

  3. Clear the check boxes of all messages you want to add to the Log Ignore List.

  4. Click OK to close the Project Settings dialog.

    The next time a log is generated, the cleared message types will be ignored by Wwise and will not be displayed in the log.

    To remove a message type from the Log Ignore List, select the check mark for the corresponding message type.

    Note

    You can also add messages to the Log Ignore List by right-clicking one or more messages directly in the log and selecting Add Message(s) to Ignore List.

To change the severity of a message:

  1. From the menu bar, click Project > Project Settings.

  2. In the Project Settings dialog, select the Logs tab.

  3. Change the severity using list in the Severity column.

  4. Click OK to close the Project Settings dialog.

    The next time a log is generated, the selected severity will be applied.

To limit the number of messages that will appear in the logs:

  1. From the menu bar, click Project > Project Settings.

  2. In the Project Settings dialog, select the Logs tab.

  3. Enable the Limit number of messages displayed to: option and type the maximum number of messages you want displayed in the log. Note that this number is used to limit all message types in the log.

  4. Click OK to close the Project Settings dialog.

Note

Manage messages that appear in the General, Project Load, and WAAPI logs through the Logs Settings dialog, which is opened by clicking the settings icon within the Logs view.

Ignore

Determines whether a message type will be displayed in the logs. By default all message types will be displayed. When this check box is selected, the corresponding message type will be ignored by Wwise and will not be displayed in the logs.

ID

The ID assigned to a particular error, warning, or message.

Message

A description of the type of issue that can be encountered during conversion or SoundBank generation.

Severity

The severity to be applied when this issue is encountered during conversion or SoundBank generation.

Note

When using the command line SoundBank generation, the WwiseConsole.exe process returns a code representing the maximum severity value found during the SoundBank generation. 0:Only Messages or no message, 1:Fatal Errors or Errors, 2:Warnings. You can use this column to control the return value.

Max Messages Per Message Id

Determines whether Wwise will limit the number of messages (per type) that will be displayed in the log. You can specify the maximum number of messages that will be displayed.

The same number will be applied to all message types.
