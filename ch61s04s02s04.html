Contents Editor: State Group

All the States within the currently selected State Group are displayed in the State Group Contents Editor. You can quickly edit several of the properties associated with each state without having to open each one in the Property Editor.

Interface Element

Description

Click the Configure Columns shortcut (right-click) option from the column header band.

The Object Property Settings opens. Specify which columns to display and their order.

Note

You cannot configure the columns for Source Contents Editors.

Name

The name and corresponding icon of the State.

Notes

Additional information about the State.
