Auto-ducking tab

In the Auto-ducking tab for Audio Busses, you can make the current bus more prominent by inserting other busses in the pane. The volume of all inserted busses is lowered when there is a signal on the current bus.

For detailed instructions, refer to Ducking signals. For a description of all interface elements, refer to Auto-Ducking tab: Audio Busses.

See Defining the properties of a bus.

The following figure shows the Auto-ducking tab for an audio bus.

Current bus: In this example, the Dialogue bus. When there is a signal on this bus, ducking occurs on the other busses, at item 2.

Busses controlled by the current bus: The volume of these busses is lowered.
