States tab

The States tab allows you to add State Groups, which contain States, to the current object. You can then customize additive type properties, such as pitch and volume, for each of the States to further define the characteristics of the object. Add or remove properties, which vary somewhat according to the selected object's type, by clicking the Properties button.
