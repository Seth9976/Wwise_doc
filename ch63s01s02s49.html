Could not connect to audio input device. Audio input callback will not be called.

The Audio Device was unable to connect the callbacks to the hardware audio input device. On iOS, input callbacks must be set up by the System Audio Device to be able to consume microphone input.

Probable causes:

  • The hardware device that is configured for audio input cannot emit a mono signal.

Recommended resolution steps:

  • Configure a different audio input device in the iOS system settings.

  • If this error persists with multiple input devices, contact Audiokinetic Support.
