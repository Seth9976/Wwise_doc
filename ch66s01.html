AK ASIO Input

Caution

This plug-in is still experimental.

(See the AK ASIO input properties below.)

The AK ASIO Input plug-in can be used to input sounds from an ASIO compatible device.

Caution

AK ASIO Input requires an ASIO Audio Device (AK ASIO Output) to be running in order to capture inputs from this device.

The AK ASIO Input plug-in must be installed using the Launcher. For information on installing plug-ins, see Installing Plug-ins in the Audiokinetic Launcher guide.

For information on creating sources using plug-ins, see Creating sources using plug-ins.

AK ASIO input properties

Interface Element

Description

AK ASIO Input Properties

Channel Configuration

Sets the number of channels as well as the configuration of the input source. See Understanding Bus Configurations for more information.

Base Channel

Offsets the input channels so the first channel used as an input is the base channel.
