Copy States Values

The Copy States Values dialog allows you to copy the property settings from one custom State to another. You can copy the State values to an existing or newly-created State within the same State Group.

Interface Element

Description

State Group

The name of the State Group

(State Group Selector)

Displays a complete list of State Groups in the project.

When you select a State Group, the name is displayed in the State Group field.

From

The name of the source State whose values you want to copy.

(State Selector)

Displays the list of States within the selected State Group.

When you select a State, its name is displayed in the From field.

To

The name of the target State to which the State values will be copied.

(State Selector)

Displays the list of States within the select State Group.

To create a new State within the selected State Group, click the 'New' option.

When you select a State, its name is displayed in the To field.

Affected objects

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Use

Determines which objects will have the State values copied.

Object Name

The names of the objects in your project that already subscribe to the State Group selected.

The objects in this list are basically the ones that will be affected by the newly copied State values.

Action

Specifies the Action that will be taken when the State values are copied to the State selected in the 'To' field. The following two actions are possible:

  • Overwrite existing state values - When the object already uses custom state properties, these properties will be overwritten with the copied property settings.

  • Create new state values - When the object uses the default state properties, a custom version will be created with the copied property settings.

Selects all objects in the list.

Clears the selection for all objects in the list.

Copies the custom property settings from the source to the target State.

Closes the dialog without copying the custom property settings.
