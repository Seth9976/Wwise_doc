Contents Editor: Switch Group

The Switch Group Contents Editor displays the list of Switches within the selected Switch Group. It also allows you to quickly rename and add notes for all the Switches within a Switch Group.

Interface Element

Description

Click the Configure Columns shortcut (right-click) option from the column header band.

The Object Property Settings opens. Specify which columns to display and their order.

Note

You cannot configure the columns for Source Contents Editors.

Name

The names and corresponding icon of the Switches within the selected Switch Group.

Notes

Additional information about each Switch.
