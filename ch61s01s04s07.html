Contents Editor: Music Track

This Contents Editor gives you quick access to the properties associated with the audio source.

Interface Element

Description

Click the Configure Columns shortcut (right-click) option from the column header band.

The Object Property Settings opens. Specify which columns to display and their order.

Note

You cannot configure the columns for Source Contents Editors.

[name]

The name of the object.

Audio File

The name of the audio file to which the source is linked.

Make-Up Gain

An offset value, either an increase or decrease, that is applied to the volume of the audio file.

Default value: 0
Range: -24 to +24
Units: dB

This option is not available in the Music Sources Contents Editor.

Duration

The duration of the audio source.

Notes

Any additional information about the object properties.
