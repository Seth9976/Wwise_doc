Import Completed

The Import Completed dialog displays the number of files that have been successfully imported, as well as information about the import process, such as the import status or any import errors.

Interface Element

Description

Import

A double progress bar that displays the progression of the media files that are being imported and a message that displays the number of media files that have been successfully imported.

Messages

Provides additional information about errors during the import; for example, an imported file may be missing some component.

Copies the information in the message to the Windows clipboard so that you can paste the information into a new file.

Closes the Import Completed dialog and then continues the SoundBank generation process.

Closes the Import Completed dialog.
