VST3 plug-ins

The following Wwise plug-ins are available in VST3 format:

  • Compressor

  • Stereo Delay

  • Expander

  • Flanger

  • Gain

  • Guitar Distortion

  • Harmonizer

  • Matrix Reverb

  • Parametric EQ

  • Peak Limiter

  • Pitch Shifter

  • RoomVerb

  • Stereo Delay

  • Tremolo

These plug-ins are compatible with any DAW that supports the format and can therefore be used in DAWs as well as in Wwise. While these plug-ins are available in Wwise by default, installation is required to use them in a DAW.

Installing the Wwise VST3 plug-in suite

To download and install the VST3 plug-ins, you must have an Audiokinetic user account.

To install the Wwise VST3 plug-in suite:

  1. Log in to your user account on the Audiokinetic website.

  2. Click your name and then Account & Settings.

  3. Under the Downloads section, accept the required legal document.

  4. Download the Wwise VST Plugin package according to your operating system (Windows or macOS).

  5. Do one of the following, depending on your operating system:

    • Windows: Extract the contents of the downloaded .zip archive and move the .vst3 files to the following directory: C:\Program Files\Common Files\VST3. If the VST3 folder doesn't exist on your computer, create it.

      Windows might ask for permission to copy files to this folder. Accept the request.

    • Mac: Double-click the downloaded .pkg file. An installer opens. Follow the on-screen instructions to complete the installation. The installer automatically moves the .vst3 files to the correct location.

Feature limitations

Certain plug-in features are unavailable in the VST3 versions of the Wwise plug-ins. All controls and options related to named channel configuration are disabled, primarily because of the anonymous nature of the channel configuration in VST3 plug-ins.
