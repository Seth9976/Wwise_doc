Common tabs and categories: Containers hierarchy objects

Properties in Wwise are grouped into logical tabs in the Primary Editor and logical categories in the Property Editor. Both editors are found in Object Tabs. The following tabs and categories are common across all Containers hierarchy objects:

  • MIDI category: Containers hierarchy objects to define the MIDI synthesizer properties of objects in the Containers hierarchy.

  • Sources category: Sound SFX and Sound Voice to modify the properties of the sources of a sound object.

Also see Common tabs and categories: audio structures for other tabs and properties common across multiple Containers hierarchy objects.
