Contextual Help

The Contextual Help view provides help entries based on your interactions with the Wwise user interface. This includes information related to the following:

  • The most recently selected property within Wwise. A help entry appears if the selected field, check box, slider, or button sets a property, but does not appear if the selected element opens a dialog, list, or view.

  • The selected error message in the Capture Log. The same information is available in Errors Reported in the Capture Log.

You can set the Contextual Help language in the Help menu.
