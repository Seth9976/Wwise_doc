Master Volume

The Master Volume module increases/decreases the gain of all audio channels by the specified decibel value. It is also possible to specify separate gain values for each channel.

A different gain value can be specified for each channel, for a channel configuration of up to 7.1.4. Active channels are identified in color when profiling is enabled.

A mute and solo button can be set for each channel for auditioning purposes.

The Link button in the upper-right corner of the module allows you to move all faders together, relative to their initial position.
