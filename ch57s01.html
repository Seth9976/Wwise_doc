Control Surface Devices dialog

Use the Control Surface Devices dialog to configure the hardware controller devices to use with Wwise.

Tip

If a MIDI device is disconnected or turned off while Wwise is open, you can open the Control Surface Device dialog to reconnect them automatically.

Interface element

Description

Columns

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Device Name

Displays the name of the control surface device.

This name is used in Control Surface Bindings.

Device Type

Specifies the type of device.

Receive From

Defines the input device to use.

Status
(Receive From)

Shows the connection status of the input device.

Send To

Defines the output device to use.

Status
(Send To)

Shows the connection status of the output device.

Buttons

Add

Adds a new device to Wwise.

After clicking Add, enter a relevant name for the device and click OK.

Remove

Removes the selected device from the device list.

Rename

Changes the device name.

Caution

Changing the device name breaks existing Bindings that use that device name.

Close

Closes the dialog.

Note

The devices specified in the list are automatically saved.
