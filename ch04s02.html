Understanding file length limitations and naming conventions

Your operating system might restrict the maximum length of a combined filename and file path. For example, the limit is 260 characters on Windows 10 and 1024 on macOS. A file with a combined name and path at or near the maximum length might be successfully imported into Wwise but then produce errors during subsequent conversion.

When naming projects and objects in Wwise, you can use any Unicode-supported character.

For the following elements in Wwise only unaccented Roman letters and underscores are accepted. Numbers can be used but not as the first character. This is to comply with the naming restrictions of certain game engines.

  • SoundBanks

  • Events

  • Dialogue Events

  • Effect ShareSets

  • Switch Groups

  • Switches

  • State Groups

  • States

  • RTPC Game Parameters

  • Triggers

  • Work Units
