Too many game objects in array in <Function>

The game object parameter array of the specified function is too large to fit in the command queue.

Recommended resolution steps:

  • Reduce the number of game objects used in this function.

  • Enlarge the command queue memory by changing AkInitSettings.uCommandQueueSize before AK::SoundEngine::Init. See Advanced Sound Engine Integration for more information on initialization settings.
