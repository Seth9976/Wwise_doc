Profiler Statistics (Query - Dialogue Events Distribution)

The 'Dialogue Events Distribution' query displays statistical information about the Dialogue Events that were triggered during the capture session. Specifically, it shows you how many times each Dialogue Event was resolved versus how many times it actually resulted in the playback of an audio structure. You can drill down even further for each Dialogue Event to find out which paths were resolved, and how many times they were resolved, both in terms of actual numbers and as a percentage of the total. Several filters also exist allowing you to sort through the information quickly to determine if any actions are necessary.

Interface Element

Description

Filters

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Filter

The types of filters you can use to filter the information found for the selected query. You can filter the information using any of the following filters:

  • Event Name - Displays information for a particular Dialogue Event.

  • Added - Filters information based on the number of times a Dialogue Event was added to the dynamic sequence.

  • Played - Filters the information based on the number of times a Dialogue Event resulted in the playback of an audio structure.

Parameters

A series of properties that allow you to define the specific criteria for each filter.

For fields that accept text strings, you can type a portion of the name; however, wildcard symbols are not supported.

Results

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Event Name

The name of the Dialogue Event.

Added

The number of times the corresponding Dialogue Event was resolved and an audio structure was added to the dynamic sequence list.

Played

The number of times the Dialogue Event resulted in the playback of an audio structure.

Selected Event

The Event selected in the Results list for which you want further statistical information.

Filters

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Filter

The types of filters you can use to filter the information about the selected Dialogue Event. You can filter the information using any of the following filters:

  • Value - Displays information related to a particular State Group or State.

  • Count - Filters information based on the number of times the path was resolved.

  • % - Filters the information based on the actual percentage the path was resolved.

Parameters

Specific properties that allow you to define the specific criteria for each filter.

For fields that accept text strings, you can type a portion of the name; however, wildcard symbols are not supported.

Value Distribution

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Value

The name of the State Group and/or states associated with the selected Dialogue Event that were posted during the current capture session.

Count

The number of times the corresponding state has been posted in the current capture session.

%

The number of times the corresponding state has been posted represented as a percentage of the total number of states posted during the current capture session.
