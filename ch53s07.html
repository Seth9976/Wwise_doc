Opening files in an external editor

You can open files in the Results pane in an external editor in the following ways:

  • Drag and drop: Select one or more files from the Results pane and drag them into the external editor.

  • Right-click: Right-click one or more files, select Open in External Editor, and then select your editor.

    Note

    If you haven't added an external editor to Wwise, select Open in External Editor > Manage External Editors. See Managing external editors for audio.

  • Ctrl+E: Select one or more files and then press Ctrl+E to open the selected files in your default external editor.
