Loudness Meter

The Loudness Meter provides a standard way to measure the loudness level and loudness range of an audio signal. It is based on the following standards:

  • EBU R128

  • ITU-R BS.1770-4

For more information about the loudness, refer to Monitoring the signal level.

The Loudness Meter displays the following elements:

Interface Element

Description

Momentary

Loudness level calculated using a sliding rectangular time window of length 0.4 seconds. The measurement is not gated.

Depending on the Show Live Data status on the toolbar, the momentary value can be the current measured value or the value at the time cursor position.

Short-term

Loudness level calculated using a sliding rectangular time window of length 3 seconds. The measurement is not gated.

Depending on the Show Live Data status on the toolbar, the short-term value can be the current measured value or the value at the time cursor position.

Momentary and short-term graph

Graph of the momentary and short-term values. The graph is only active when the profiler Capture is on. Use the Start Capture button on the Wwise toolbar to start collecting momentary short-term values.

When a capture session is present, it is possible to click the graph to inspect the momentary and short-term values in the past. Changing the time cursor will also affect other profiling views.

Measure

Starts or stops the integrated loudness measurement. When activated, the integrated loudness and the loudness range are being measured. The Measure status does not have an impact on the momentary or short-term loudness.

Reset

Resets the current measurement. This is equivalent to clicking Measure twice to start and stop the measurement. Resetting the measurement resets the integrated loudness and the loudness range.

Integrated

Loudness level calculated during the measurement time. The integrated loudness uses gating (all values below -70 LUFS are ignored) and is a long term measurement (an integration over the entire measurement time) based on the momentary loudness. The measurement starts when clicking the Measure button, and stops when clicking it again. Clicking the Reset button restarts the measurement.

Loudness Range (LRA)

Loudness range is a statistic calculation of the material's dynamic range corresponding to the integrated loudness measurement using gating (all values below -70 LUFS are ignored). Loudness range measures the variation of loudness on a macroscopic time-scale, in units of LU (Loudness Unit).

The measurement starts when clicking the Measure button, and stops when clicking it again. Clicking the Reset button restarts the measurement.
