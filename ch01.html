Wwise system requirements

Before you install Wwise, ensure that your system meets the minimum requirements. The following table describes the hardware and software required to install and run the Wwise authoring application.

System Component

Requirement

Operating System

This version of Wwise was tested on the following operating systems:

  • Windows 11 64-bit

  • macOS Sequoia

If you are using a more recent operating system and you experience issues with Wwise, we recommend that you upgrade to the latest available version of Wwise.

The Wwise authoring application is only available in 64 bit.

Memory

4 GB of RAM

Resolution

The factory layouts are optimized for monitors with a resolution of 1920 x 1080, but they can be adapted for other resolutions. To support 4k or 5k resolution, enable the Enable High DPI option in the Wwise User Preferences.

PDF Reader

Adobe Reader or another PDF reader is required to open the PDF files that are distributed with Wwise.

Mandatory Code Dependencies

Microsoft .NET Framework

Microsoft .NET Framework 4.8

The Audiokinetic Launcher automatically installs this component before it installs Wwise. However, if you deploy Wwise without the Launcher (through Perforce or some other source control system), you must install the .NET Framework manually in order to run the File Packager. Refer to the Microsoft .NET Framework download page.

Note

Crashes can occur due to one or more missing dependencies when you install Wwise without the Launcher. The required dependencies are detailed in the following rows of the Wwise System Requirements table: Microsoft .NET Framework. We recommend that you use the Launcher to install the Authoring tool to ensure that the dependencies are installed at the same time.
