Audio Object 3D Viewer Options

The Audio Object 3D Viewer Options allow you to select the options that you want to appear in the Audio Object 3D Viewer.

Interface Element

Description

General

Show Grid

Shows the grid lines and numbers in the 3D view.

Linear Threshold (meters)

Below this distance from the origin, the grid is presented in a Linear scaling fashion, and beyond this distance the grid is Logarithmic. These two scaling methods function as follows:

  • Linear Scaling: Shows the Audio Objects with their position scaled linearly from the listener. This scaling is useful if the Audio Objects are positioned within a limited range to the listener. As a reference for distance, a representation of a human head is included, the size of which can be configured using the Inter-Ear Distance (cm) setting.

    Note

    When the game connects the scaling factor of game units per meter will be automatically set in the view settings. In order to get an accurate head scaling, ensure the game units per meter value is set correctly.

  • Logarithmic Scaling: Shows the Audio Objects with their position scaled logarithmically from the listener. This scaling is useful if the Audio Objects are positioned in a wide range, such as an exterior world.

Grid Max (meters)

Maximum value for all Audio Objects to be shown. All Audio Objects outside this range will be shown outside the grid range.

Grid Size

In the logarithmic portion of the grid (beyond the Linear threshold) this visually scales the grid up and down. This has an effect that is very similar to zooming. In the linear portion of the grid (within the Linear Threshold), the grid does not scale, nor do the head and sphere.

Grid Units

Allows the user to select how the grid numbers (distances) are presented. The available choices are Meters and Game Units. The Game Units To Meters setting specifies the ratio between the two.

Show X-Y-Z Axis

Shows the three axes in the lower left corner. These correspond to the listener coordinate space.

Inter-Ear Distance (cm)

Sets the distance between the ears of the head in linear scaling mode.

Note

Audio Object distances shown in relation to the head must be in the same units. Use the Game Units Per Meter setting in order to change the number of game units in a meter.

Game Units Per Meter

Sets the scaling factor of the game units per meter for the view. For example, for game units that are in centimeters, this value will be 100.

You don't need to adjust this setting if it is set by your game with the fGameUnitsToMeters initialization setting.
