Early reflections are not supported on sounds using 3D Position: Listener with Automation. The assigned early reflections bus will be ignored.

When using an early reflections bus, a sound's 3D Position must be set to either Emitter or Emitter with Automation. However, note that when using Emitter with Automation, reflections are still only calculated from the emitter position. For the source of reflections, the automation offset is ignored. For more information about setting up early reflections on a sound, see Spatial Audio.

Recommended resolution step:

  • In the General settings tab of the sound, clear the Early Reflections Auxiliary Bus (that is,set it to None), or change the 3D Position to Emitter in the Positioning tab.
