Contents Editor: Sine

The Contents Editor gives you quick access to some of the most common properties associated with the Sine plug-in. You must click the Sine source to display the associated column headings.

Interface Element

Description

Name

The name of the sine plug-in.

Use

Determines which source within the sound object is:

  • Played during play back.

  • Included in SoundBanks.

  • This option is only visible when you have more than one source within the sound object.

Frequency

The frequency of the tone.

Default value: 440.0

Range: 20 to 20000

Units: Hz

Gain

The level or amplitude of the tone’s output.

Default value: -12

Range: -96.3 to 0

Units: dB

Duration

Duration of the tone where the amplitude is constant over time.

Default value: 1.0

Range: 0.001 to 3600

Units: s

Notes

Any additional information about the Sine plug-in.

Adds a new source to the sound object.

When you click the Add Source button a list of possible Source plug-ins that can be added is displayed.
