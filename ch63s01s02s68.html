Failed creating source.

Generic error indicating that an audio source could not be started.

Probable causes:

  • There is a non-registered plug-in used by the source. There was likely a plug-in error preceding this error.

  • Not enough memory to create the source (would be preceded by an out of memory error)

  • The bus for this source doesn't exist. Maybe the Init SoundBank is out of date.

  • An unforeseen error.

Recommended resolution steps:

  • Check for out-of-memory errors in the log and resolve them by allocating more memory. See Memory allocation failed..

  • Ensure the Init SoundBank is in sync with the SoundBank containing this source. Regenerate and recopy the SoundBanks.

  • Contact Audiokinetic Support. Ideally, capture a ProfilingSession.prof while reproducing the problem. (Refer to Saving the Capture Log for details.)
