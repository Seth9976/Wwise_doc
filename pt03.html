Interacting with the game

This section includes conceptual and procedural information on how to use events, dynamic sequences, and Wwise game syncs to drive the sounds in your game.

  • Managing Events—creating Events, adding Actions and assigning their Targets, setting properties and defining the scope of an Event Action.

  • Managing dynamic dialogue—understanding the Dynamic Dialogue system, working with Dialogue Events and States,creating paths and fallback paths, working with probability and weighting.

  • Working with States—creating State Groups, defining Transitions between States, and assigning States to objects and busses.

  • Working with Switches—creating Switches and Switch Groups, mapping Game Parameter values to Switches, and assigning objects to more than one Switch Group.

  • Working with RTPCs—managing Game Parameters used in RTPCs, controlling property values using Game Parameters, working with the RTPC graph view, building smart pitch curves, working with LFOs, envelopes, and time, and setting up side-chaining.

  • Working with Triggers—creating Triggers and understanding the Trigger mechanism.

  • Working with States and State Groups for dynamic dialogue—creating States and State Groups for use with Dynamic Dialogue, including a practical example.
