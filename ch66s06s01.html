Source Editor: Haptic Clip Player

The Source Editor displays all the properties associated with the Haptic Clip Player plug-in.

Note

The frequency envelope of a .haptic clip can end before the amplitude envelope. When this occurs, the plug-in holds the last frequency value for the remainder of the clip. The clip ends when the amplitude envelope ends.

Interface Element

Description

Haptic File

Haptic File

The path to the .haptic file used as the source for this plug-in. The file is stored as a reference and converted to an optimized binary format during SoundBank generation. All amplitude and frequency values must be in the range [0.0, 1.0] and control points must be in chronological order; clips that fail validation will not produce output.

Click Load to browse for a .haptic file. The file must conform to the Meta Haptics SDK specification.
