Schematic View Settings

The Schematic View Settings dialog allows you to filter the amount of information related to property values appearing in the Schematic view.

Interface Element

Description

General

Icon Strip

Displays the row of icons representing changes that have been made to object properties.

Mute/Solo

Displays the Mute and Solo buttons for the object.

Bus

Displays the bus routing for the object.

Conversion Settings

Displays the Conversion Settings ShareSet that is applied to the object, if any.

Effect

Displays the Effects applied to the object, if any.

Game Parameters

Displays any Game Parameters that affect the object through RTPCs.

State Group

Displays any State Groups that affect the object.

Advanced Settings

Displays any advanced settings that have been specified for the object.

Properties

Volume

Displays the volume setting for each project object.

Pitch

Displays the pitch setting for each project object.

Low-Pass

Displays the voice Low-Pass Filter setting for each project object.

High-Pass

Displays the voice High-Pass Filter setting for each project object.
