Reference topics

This section provides complete information about all the options and views in Wwise, and is accessible through the Help icon in the upper-right corner of views, dialogs, and so on.

  • Project—reference material covering all functionalities accessed through the Project menu.

  • Views—reference material covering all functionalities accessed through the Views menu.

  • Edit, Windows, and Help—reference material covering all functionalities accessed through the Edit, Audio, Windows, and Help menus.

  • Project Explorer—reference material covering all functionalities accessed through the tabs of the Project Explorer, including Work Units, Presets, Randomizer, Batch Rename, List view, Multi Editor, Schematic view, Reference view, and the Object Property Settings dialog.

  • Event Viewer—reference material on the Event Viewer and its three tabs: Filtered, Current Selection, and Orphans.

  • Profiler Views—reference material covering the many profiler views: Capture Log, Performance Monitor, Advanced Profiler, Voice Inspector, Audio Object 3D Viewer, Audio Object List, Audio Object Metadata, Audio Object Meter, Voice Monitor, Voice Explorer, Game Object 3D Viewer, Game Sync Monitor, and Profiler Statistics.
