Import Definition Log

The Import Definition Log dialog displays information about SoundBanks that were imported using a definition file. The log creates an entry for each activity performed. Entries are added to the log when new SoundBanks are created, when Events are added, deleted or are detected missing from existing SoundBanks, and when game sync exclusions are added, deleted, or reported missing. If there are no changes to a SoundBank, the log displays a “No Change Detected” message.

Interface Element

Description

SoundBank

The name of the SoundBanks that were imported.

Import Activity

Displays the activity that occurred during import:

  • Event Added: A new Event has been added to the existing SoundBank.

  • SoundBank Created: A new SoundBank has been created.

  • Event Deleted: An Event has been deleted from the existing SoundBank.

  • Event Missing: An Event no longer exists in the project or has yet to be created.

  • Exclusion Added: A game sync was excluded from the existing SoundBank.

  • Exclusion Deleted: A game sync was re-included into an existing SoundBank.

  • Exclusion Missing: A game sync no longer exists in the project or has yet to be created, so the exclusion could not be added or deleted from the exclusion list.

  • No Change Detected: The imported SoundBank is identical to the one already in Wwise.

Event

The name of the Events that have been added or deleted from the SoundBank, or are missing from the project.

Copies the information in the log to the Windows clipboard so that you can paste the information into a new file.

Closes the Import Definition Log dialog.
