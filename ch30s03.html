Associating sub-tracks with Switches and States

To add variety to your score, you can determine which sub-tracks are played via a Switch/State Group. To do so, the Track Type must be set to switch. For more information about how to work with behaviors and tracks, refer to Defining the playback behavior of Music Tracks.

To associate a sub-track to a Switch/State, a Switch/State Group must be selected for the track. The track's Switch/State Group is selected via the controls in the Switch Type box of the track's Property Editor.

The behavior of the track upon a Switch/State change is set via the Transitions tab. For more information on how to work with transitions, refer to Working with transitions.

To associate a sub-track to a Switch/State:

  1. Right-click the sub-track's header (yellow bar).

  2. In the shortcut menu, select Association.

  3. Select either an existing Switch/State, or create a new one.

    The sub-track is associated with the Switch/State.

    The Switch/State name is displayed along the top of the sub-track.
