SoundBanks Settings - External Sources Tab

The External Sources tab of the SoundBanks Settings dialog is where you can specify custom settings for the external sources used by the External Source plug-in. You can override the project settings and use your own External Sources List file. You can also specify a different output folder where the external source audio files are saved for use in your game.

Interface Element

Description

Convert external sources when generating banks

If enabled, External Source audio files are converted to WEM format as part of the SoundBank generation process. The files must be in WEM format to be used by the sound engine at runtime; however, if you’re not using the files locally, you can disable this option to speed up SoundBank generation.

Default value: true

Override Input Path

Overrides the project settings, which allows you to use a different External Sources List file.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Platform

The names of the various platforms that are currently active in your project.

External Sources List

The full path where the External Sources List file is located.

This data file contains information about the external sources that will be used by the External Sources plug-in.

(Browse)

Opens a Windows browser where you can navigate to the folder where the input audio source list is located.

Override Output Path

Overrides the project settings, which allows you to specify your own custom output folders for the converted audio files used by the External Sources plug-in.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Platform

The names of the various platforms that are currently active in your project.

External Sources Output Folder

The folder where the converted external sources are output.

(Browse)

Opens the Browse For Folder dialog where you can navigate to the folder where the converted external audio sources will be saved.

Closes the Project Settings dialog and saves your settings.

Closes the Project Settings dialog without saving your settings.
