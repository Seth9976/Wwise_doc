Including plug-in media in SoundBanks

Some effects, such as the AK Convolution Reverb, require media files to work. For example, the AK Convolution Reverb requires an impulse response file as its plug-in media. Those plug-in media files need to be included inside a SoundBank for the Effect to function normally in the game. You also need to make sure the SoundBanks containing the media files are loaded at the moment you need the Effect to be instantiated.

There are multiple ways of including the plug-in media files in a SoundBank:

  • You can insert the ShareSet effect object inside the inclusion list of a SoundBank.

  • You can insert the owner object (bus, Auxiliary Bus or Containers hierarchy object) inside the inclusion list of a SoundBank.

  • You can insert the Event inside the inclusion list of a SoundBank, if the Effect is owned by an Containers hierarchy object.

It is possible to verify that the plug-in media files were included in a SoundBank by looking at the SoundBank Editor/Edit tab.
