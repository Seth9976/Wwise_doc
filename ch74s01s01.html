Multi-Channel Creator window

In the Multi-Channel Creator window, you can specify the location of the source mono and/or stereo files that will be used to create the multi-channel files that will be imported into your Wwise project. You can also specify the output folder where the newly-created multi-channel files will be saved. As soon as you specify an Input folder, the Output preview gets populated automatically. The Output preview displays the multi-channel files that will be created along with a list of possible output configurations.

For more information about channel configurations, refer to Understanding Bus Configurations.

Interface Element

Description

Input folder

The folder that contains the source mono and stereo audio files that will be combined to create your multi-channel files.

The name of the each audio file in the Input folder must contain a specific suffix so that the Multi-Channel Creator can easily identify which channels exist for each file. You can define these suffixes in the Settings dialog.

Opens the Browse for Folder dialog where you can navigate to the 'Input folder'.

Output folder

The folder where the multi-channel files will be saved.

Opens the Browse for Folder dialog where you can navigate to the 'Output folder'.

Output Preview

(Selection)

Determines which multi-channel files will be generated.

Output file name

The name of the multi-channel files that will be generated.

If the source files in the Input folder are organized into sub-folders, the Output file name will include the full path relative to the output folder.

Configuration

A list of possible multi-channel configurations. These configurations are based on the channels that exist for each source audio file in the Input folder.

For more information about channel configurations, refer to Understanding Bus Configurations.

Selects all multi-channel files in the Output preview list for processing.

Clears the selection of all multi-channel files in the Output preview list.

Starts the generation process for the selected output files.
