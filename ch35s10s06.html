Conversion - In Progress/Stopped

The Conversion - In Progress dialog contains a primary progress bar that displays the overall progress of the audio source conversion as well as a set of secondary progress bars for each file that is being converted. The number of secondary progress bars will depend on the number of processing cores on your machine. Wwise uses each processing core to make the conversion process as fast as possible. The dialog also displays a log that contains information about the conversion process including the status of the conversion and any errors.

You can stop the conversion process at any point. When the conversion is interrupted, Wwise automatically displays the Conversion - Stopped dialog.

Interface element

Description

Operation

Displays the name of the high-level task being performed along with the name of each file that is being converted.

Progress

A series of progress bars that displays the progression of the media files that are being converted.

A primary progress bar displays the overall progress of the audio source conversion. One or more secondary progress bars display the progress of each media file as it is being converted. If your machine has a multi-core processor, each core will have its own progress bar.

This column is empty when the conversion is stopped.

Details

Displays detailed information about the particular task being performed by each core.

This column is empty when the conversion is stopped.

Log

(Type - Color)

Displays the color associated with the different types of messages. The following types exist in the Log:

  • Yellow - For conversion warnings.

  • Red - For conversion errors.

  • White - For general messages.

Time

The time the warning, error, or message was generated.

ID

The ID assigned to the particular error, warning, or message.

Message

A description of the issue that was encountered while the file was being converted.

Platform

The platform on which the issue was found.

Parameters

A list of project elements that are related to the error, warning, or message.

You can double-click a project element to load it into its editor.
