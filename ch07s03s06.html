Specifying project settings for External Sources

If you plan to use the External Source plug-in you must specify the location of the External Sources List file. This is an XML file that specifies:

  • The location of the external audio files that can be associated with the External Source you created in Wwise.

  • The Conversion Settings that will be used to convert each file.

You must also specify the folder in which the converted sources will be saved so that they can be used by the Wwise sound engine at runtime. The output folder path can also include environment variables as part of the path, for example, $(WWISESDK) or $(TEMP).

You can override these project settings with the user settings of the SoundBank Manager. See Specifying user settings for external sources.

To specify project settings for external sources:

  1. From the menu bar, click Project > Project Settings.

  2. In the Project Settings dialog, switch to the External Sources tab of the Project Settings dialog.

  3. In the Input Path group, specify a folder where the External Sources List file (WSOURCES) is located by doing one of the following:

    • Click in the External Sources List and type the path where the External Sources List file is located.

    • Click the browse button […] that corresponds to one of the game platforms, navigate to the folder that contains the External Sources List file for that platform, and then click Open.

  4. Repeat step 3 for each of the active platforms in your project.

  5. In the Output Path group, specify a folder where the converted external sources will be saved by doing one of the following:

    • Click in the External Sources Output Folder list and type the path where you want the audio files to be saved.

    • Click the browse button […] that corresponds to one of the game platforms to use the Advanced Folder Picker to specify the folder where you want to save the converted audio files for that platform, and then click OK.

  6. Repeat step 5 for each of the platforms.

  7. Click OK to close the Project Settings dialog.

External Sources tab of the Project Settings dialog

In the External Sources tab of the Project Settings dialog, you can specify the location of the External Sources List file. This XML file contains information about the audio files used by the External Source plug-in, including their location, conversion settings, and output folder where the external source audio files are saved for use in your game.
