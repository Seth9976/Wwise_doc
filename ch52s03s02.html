RTPC tab

The controls on the RTPC tab enable you to edit specific sound properties in real time based on real-time parameter value changes that occur within the game. The parameter values are displayed in a graph view, where one axis represents the property values in Wwise and the other axis represents the in-game parameter values. You can focus on one curve at a time or edit all curves simultaneously.
