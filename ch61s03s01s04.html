SoundBank Editor - Details Tab

The Details tab of the SoundBank Editor displays information about the size of the different elements within the selected SoundBank as well as any files that may be missing. The information on this tab is sorted by language.

Interface Element

Description

Name

The name of the SoundBank.

Notes

Any additional information about the SoundBank.

Maximum size

The maximum amount of in-game memory that you allocated for a particular SoundBank.

Units: Bytes

Date updated

The date and time the SoundBank was created or last updated.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Types

The different types of objects that are included in the corresponding SoundBank, for example, SFX, Voice, and Music. If the SoundBank contains all four types of objects, the word “All” is displayed in this column.

Languages

A list of languages used in the current project. If more than one language is used, they will appear in alphabetical order.

If no Voice objects are included in the SoundBank, this field will be empty.

SFX

The amount of memory occupied by the Sound SFX objects in the current SoundBank.

Units: Bytes

Music

The amount of memory occupied by the Music objects in the current SoundBank.

Units: Bytes

Voice

The amount of memory occupied by the Sound Voice objects for a particular language.

Units: Bytes

Data Size

The amount of memory occupied by the Sound SFX and Sound Voice objects for a particular language.

The current size will appear in red if it exceeds the maximum allocated for this SoundBank.

Units: Bytes

Note

You can successfully generate a SoundBank even though it exceeds the maximum memory that you allocated.

Free Space

The amount of space remaining in the SoundBank. Free space is calculated by subtracting the Data Size from the maximum size. If the Data Size exceeds the Maximum size, the Free Space value will be displayed in red.

Units: Bytes

Missing Files

The number of media files, both Sound SFX and Sound Voice, missing from a particular language.

Files Replaced

The number of missing Sound Voice audio files in a particular language that are currently replaced by the audio files of the reference language.

Memory Size

The amount of space occupied by the SoundBank data that is to be loaded in memory.

Units: Bytes

Prefetch Size

The amount of space occupied by the prefetch data of the streamed media files. This information is loaded into memory when the SoundBank is loaded to ensure that the streamed media files playback without delays.

Units: Bytes

File Size

The total size of the generated SoundBank file.

Units: Bytes
