Build Smart Pitch Curve

In the Build Smart Pitch Curve dialog, you can define the properties of a smart pitch curve. Smart pitch curves are RTPC curves automatically generated by Wwise from values you provide. These curves represent natural-sounding variations in pitch caused by changes in a game property value.

Interface Element

Description

Native (X Axis parameter) value

The Game Parameter value at which the sound is heard at its original pitch.

Subdivision level

The precision of the curve, graded from 1 to 10.

Creates a smart pitch curve based on the native value and subdivision level.

Closes the dialog without building a smart pitch curve.
