Dynamic Sequence ID not found.

An operation on a Dynamic Sequence was attempted on a sequence ID that was never opened. Make sure that all Dynamic Sequence function calls on that Sequence ID occur after a successful call to AK::SoundEngine::DynamicSequence::Open. You can validate the order of calls as they were received by Wwise by enabling API Calls in the Profiler Settings.
