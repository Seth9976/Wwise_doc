Profiler Views

Three layouts cover different scopes of profiling within Wwise, and each has its own specific views.

Layout

Description

Included Views

Profiler

The Profiler layout allows you to capture and monitor the performance of each game element as it occurs. You can also monitor the performance of simulations triggered by the Soundcaster all before they have been integrated in game.

  • Advanced Profiler

  • Capture Log

  • Performance Monitor

Voice Profiler

The Voice Profiler layout allows you to monitor voices captured in your game and explore the structures driving each one.

  • Capture Log

  • Project Explorer

  • Property Editor

  • Transport Control

  • Voice Inspector

  • Voice Monitor

Game Object Profiler

The Game Object Profiler layout allows you to monitor game objects registered in your game, including in their roles as emitters, listeners, or both simultaneously.

  • Game Object 3D Viewer

  • Game Sync Monitor

Complementary Profiler View

  • Profiler Statistics: Allows you to gather statistics about some audio elements in your game, after having completed a Capture Session.
