File Order Editor

In the File Order Editor, you can arrange the files within a package in a particular order. Ordering all files within a package can be time-consuming and unnecessary. If only a few of the files within the package need to be arranged in a particular order, you can order these files and then use the 'Remaining files inserted here' placeholder to position all other files within the package.

To access the File Order Editor, click the Edit file order button in the Packages view of the File Packager window.

Interface Element

Description

Package files

Filename

The name of the SoundBanks, streamed files, or loose media that are included in the package.

File Type

The type of file: SoundBank, streamed file, loose media, or a mix.

Language/SFX

The language to which the corresponding file is associated. If the file is not associated with any language, it is considered an SFX.

File size

The size of the file.

Adds the selected files to the next position in the ordered list.

You can also drag files directly from the Package files view to a specific position in the ordered list.

File order

Filename

The name of the SoundBank, loose media, or streamed file that has been arranged in a particular order within the package.

The 'Remaining files inserted here' placeholder represents all files that have not explicitly been arranged in a particular order within the package.

File Type

The type of file: SoundBank, streamed file, loose media, or a mix.

Language/SFX

The language to which the corresponding file is associated. If the file is not associated with any language, it is considered an SFX.

File size

The size of the file.

Removes the selected file(s) from the ordered list.

When a file is explicitly removed from the ordered list, it still remains in the package, but is added back to the 'Remaining files inserted here' placeholder.

Removes all files marked as 'missing' from the ordered list.
