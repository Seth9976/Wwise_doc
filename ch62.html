Event Viewer

The Event Viewer displays the Action Events that have been created for your project. It contains the following three tabs:

  • Filtered Tab which allows you to filter the list of Events by Action type.

  • Current Selection Tab which allows you to view Events for the currently selected object.

  • Orphans Tab which allows you to view Events that have been created, but not currently assigned to an object.

From within the Event Viewer, you can also browse through Events, delete Events from your project, and open the Event Editor to edit your Events.
