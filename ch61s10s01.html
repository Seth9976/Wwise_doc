New Work Unit

In the New Work Unit dialog you can specify a name for the new Work Unit that you are creating.

Caution

After a Work Unit is created, it can't be undone using undo/redo.

Interface Element

Description

Name

The name of the new Work Unit.

File

The path or location where the new Work Unit will be created.

The path includes the name and extension of the new Work Unit.

Automatically add to source control

When this option is enabled, Wwise will automatically add the new Work Unit to source control.

Note

This option is only visible when a source control plug-in has been selected in the Project Settings' General tab of the Project Settings dialog.

Creates the Work Unit in your project.

Closes the New Work Unit dialog without creating a new Work Unit.
