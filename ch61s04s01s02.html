Property Editor: Switch

The Switch Property Editor is where you can name and add notes about the selected Switch. Each Switch name must be unique and consist only of letters, digits, and underscores. The first character must be either a letter or an underscore.

Interface Element

Description

Name

The name of the Switch.

Notes

Additional information about the Switch.
