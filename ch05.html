Personalizing your workspace

There are several ways to customize the Wwise work environment. The following table summarizes the types of customization that are available.

Area of Customization

Description

Working with views

Defines the size and placement of flat and floating views.

Working with layouts

Defines which layout is displayed, which views are displayed within the layout, and where they are displayed.

Setting colors

Defines the color chip indicators of Wwise objects.

Setting user preferences

Defines your particular user preferences.

Setting authoring audio preferences

Defines your audio preferences.
