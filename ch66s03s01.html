Contents Editor: External Source

The Contents Editor gives you access to the properties associated with the External Source. Click the source entry in the Contents Editor to display the associated column headings.

For information about the SDK implementation, see Integrating External Sources in the Wwise SDK documentation.

Interface Element

Description

Name

The name of the External Source plug-in.

Use

Determines which source within the sound object is:

  • Played during playback.

  • Included in the SoundBanks.

  • This option is only visible when you have more than one source within the sound object.

Cookie

The cookie is a hash of the source name that is used at runtime to identify the external source.

Notes

Any additional information about the External Source.

Adds a new source to the sound object.

When you click the Add Source button a list of possible sources that can be added is displayed.
