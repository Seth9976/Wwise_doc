Ignoring seek after end of playlist.

A seek operation was attempted on an Interactive Music playlist for a time after the sum of the duration of all items currently in the playlist.

Recommended resolution step:

  • Using the Wwise Profiler, with API Calls enabled in the Profiler Settings, identify which SeekOnEvent API call was used with wrong values.
