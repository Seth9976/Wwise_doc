Ambisonic is only available with 3D audio. 3D audio is not enabled in the Audio Device, reverting to standard configuration.

Ambisonic mixing features can only be used when the output device is using 3D audio. On platforms that support this feature, the feature is always optional and should be enabled explicitly in the System Audio Device settings.

Recommended resolution step:

  • Enable the Allow 3D Audio option in the System Audio Device settings.
