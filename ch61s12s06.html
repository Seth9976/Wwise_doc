Object Property Settings

The Object Property Settings dialog allows you to configure which columns to show in a number of views. The available columns are sorted by groups and represent all the properties for every object type in Wwise, including some that apply to all objects.

The Object Property Settings can be opened when clicking on the View Settings icon of different views or by selecting Configure Columns when right-clicking the header band of the columns. It is also possible to open the Object Property Settings from the Hide or Show Properties shortcut menu option available in the Multi Editor.

The following views are connected, such that changes to the columns in one are applied to the others:

  • List View

  • Query Editor

  • Reference View

  • Event Editor

Note

The MIDI Keymap Editor is a view that also has its column display set with the Object Property Settings dialog; however, it is unconnected to the other views.

Similarly to the above-cited views, the Contents Editor for most Wwise objects are connected when having their columns configured. For the Contents Editor of Wwise objects, however, the Object Property Settings can only be prompted by selecting Configure Columns… when right-clicking the header band of the columns.

Note

The Contents Editor: Random Container and the Contents Editor: Switch Container can also have their column display set with the Object Property Settings dialog; however, they are unconnected to the other object Contents Editors.

Interface Element

Description

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Property

One to many tree-grouped object properties with a check box to add or remove them from the view.

Affected Object Types

A list of all the Wwise objects that can be impacted by the property, ranging from one or more object types such as the Audio Bus, containers, and sounds, or all.

Clears the selection for all properties.

Click this button to return the property selections to the default settings.
