Generating SoundBanks

The Generating SoundBanks dialogs are fundamentally the same, but they have different tag names to indicate their current status. So, while processing, the dialog is tagged "In Progress"; when stopped, a "Stopped" tag is added; and when the process completes, the tag is "Completed". So, you may start with a Generating SoundBanks - In Progress dialog and finish with a dialog entitled Generating SoundBanks - Completed.

All variations of these dialogs have three panels of information, detailed in the table below, which display:

  • the progress of different operations run during the file conversion process;

  • the results of the different SoundBanks' generation; and

  • a log of the process, including messages and warnings if something goes or could potentially go wrong.

During the process, the top panel has a primary progress bar that displays the overall progress as well as a set of secondary progress bars for each file that is being converted. The number of secondary progress bars depends on the number of processing cores on your machine. Wwise uses each processing core to convert the files in your SoundBank, making the generation process as fast as possible. The Log panel displays information about the generation process including the status of the generation and any errors.

You can stop the generation process at any point. When the generation is interrupted, Wwise automatically displays the Generating SoundBanks - Stopped dialog, which is the same dialog except that a red-font "Stopped" replaces all the operations listed in the upper panel. You can still see the Results and review the Log for any issues that appeared before stopping the generation.

Interface Element

Description

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Operation

As the SoundBanks are being generated, this column displays the name of each task that is being performed. When the generation process is complete, the word “Completed” is displayed.

Progress

As the SoundBanks are being generated, this column contains a series of progress bars that display the progression of the SoundBank generation process.

A primary progress bar displays the overall progress of the SoundBank generation. A series of secondary progress bars display the progress of each operation being performed as part of the SoundBank generation. If your machine has a multi-core processor, each core is used to convert the files in your SoundBank and will have its own progress bar.

This column is empty when the generation is complete.

Details

As the SoundBanks are being generated, this column displays additional information about the particular task being performed.

This column is empty when the generation is complete.

Results

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

SoundBanks

The name of the SoundBanks that have been generated.

Note

Init is the default name of the Initialization Bank, a special bank every project has that is listed along with all the generated SoundBanks.

Platforms

The platforms for which the SoundBanks were generated.

Language/SFX

The languages for which the SoundBanks were generated. If a SoundBank doesn't contain any Voice objects, this column displays “SFX”.

Created

Displays "Yes" or "No" to indicate whether or not the SoundBank was successfully generated. Displays "Up to Date" if the SoundBank was not generated because there were no changes since the last time it was generated.

Status

A message that describes the status of the SoundBank generation. Any irregularities with the generation, such as missing files or SoundBanks that exceed the maximum, are reported.

Log: See SoundBank Generation Tab.

Closes the SoundBanks Generation - Completed dialog.
