Positioning category

The Positioning category contains positioning properties of the selected object. You can set whether an object only uses speaker panning, 3D positioning, or a combination of them both. You can then define how each of these types of positioning is implemented. For the audio throughput of an object, you can also specify what percentage will be played through the center speaker.
