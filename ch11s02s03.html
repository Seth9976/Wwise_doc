Import Conflict Manager

In the Import Conflict Manager dialog, you can view and resolve, if possible, the following two types of errors that occur during the import process:

  • Recoverable errors that you can correct or can be corrected automatically. For example, the media file already exists, or the file name is incorrect.

  • Non-recoverable errors that require that you replace the file that cannot be imported, or make changes to the files before you can import them. Non-recoverable errors are caused by importing audio formats that are not supported by Wwise, or importing files whose properties are outside of the range specified by Wwise.

The Import Conflict Manager dialog has two sections:

  • Errors

  • Import Files

Note

Selecting the Replace operation in the Import Conflict Manager replaces the existing file. This cannot be undone, even with an Undo operation. The only way to return to the replaced file is to re-import it and replace the file with the first version of this file.

Interface element

Description

Import destination

The object into which the files will be imported.

Import language/SFX

The language specified for the files imported as Sound Voice objects. For sound SFX and music objects, SFX will be written.

Set all to

Selecting one of the command buttons applies an operation to all recoverable errors in the Error list:

  • Replace: When the media files exists, selecting this command replaces all existing media files with the imported files.

  • Use Existing: When the media files exist, selecting this command uses the existing files and not the imported files.

  • Cancel: When the media files exist, selecting this command cancels the media file import.

Note

Selecting the Replace operation in the Import Conflict Manager replaces the existing file. This cannot be undone, even with an Undo operation. The only way to return to the replaced file is to re-import it and replace the file with the first version of this file.

Errors

Audio File

The name of the media file with an error.

Status

The status of the imported file error:

  • Non-recoverable errors are displayed in red.

  • Recoverable errors are displayed in white.

  • Recovered errors are displayed as Ready in white.

Message

Provides information about the error and the possible resolution.

Operation

The list of possible operations to resolve recoverable errors.

  • Replace: When the media file exists, selecting this option replaces the existing file with the imported file. If you selected the Create new objects import mode, this option will replace existing files and create new Wwise objects.

  • Use Existing: When the media file exists, selecting this option uses the existing file and not the imported file.

  • Cancel: When the media file exists, selecting this option cancels the audio import. When the error is non-recoverable, only the Cancel operation is available.

Note

Selecting the Replace operation in the Import Conflict Manager replaces the existing file. This cannot be undone, even with an Undo operation. The only way to return to the replaced file is to reimport it and replace the file with the first version of this file.

Import Files

Audio File

Status

Displays the Ready status of the files to be imported.
