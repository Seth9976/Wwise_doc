Filtered Tab

The Filtered tab displays a complete list of Events that you have created for your project. By default, the Event list is sorted alphabetically. You can, however, sort the list by action type or filter the list to display only those Events that contain a particular action type. Like the Project Explorer, you can navigate through the filtered Events by clicking the plus (+) and minus (-) signs to expand and collapse the folders.

You can also display the Events using specific sorting criteria by clicking the following tabs:

  • Current Selection Tab

  • Orphans Tab

Interface Element

Description

Filter

Displays a list of filters that can be used to filter the Event list. Click the button to display the list of filters.

The name of the selected filter is displayed in the corresponding text box.

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

(Event list)

Displays the complete list of Events in your project. The Event list can be sorted alphabetically or according to one of the filter options.

Tip

Listed Events with "[Missing]" next to them are incomplete; they are missing a target object.
