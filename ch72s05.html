Release Notes

This section contains all of the release notes for the ReadSpeaker speechEngine plug-in.

2025.1.9

  • New Features

    • SP-6221 Added two custom demonstration voices, AWARE (robot/AI) and Sportscaster, for internal testing or evaluation of voice capabilities.

    • SP-6284 Added 17 new regional voices:

      • Katka: Czech Female

      • Lene: Danish Female

      • Ilse: Dutch Female

      • Annisa: Indonesian Female

      • David: Spanish (Latin American) Male

      • Sven: Swedish Male

      • Maja: Swedish Female

      • Ozan: Turkish Male

      • Seda: Turkish Female

      • Mai: Vietnamese Female

    • SP-6293 Added an initialization check to the speechEngine plug-in to prevent demonstration voices from loading in Release configurations. Debug or Profile configurations can still be used to test the voices.

  • Miscellaneous Changes

    • SP-6238 Add Windows VS 2026 (vc180) support.

  • Bug Fixes

    • SP-6180 Fixed: Error and status messages for ReadSpeaker voices display internal technical identifiers (such as eng_custom05) instead of the actual voice name.

    • SP-6181 Fixed: When you generate SoundBanks without a license for the ReadSpeaker (speechEngine) plug-in, Wwise displays an incorrect error message about an invalid voice license.

    • SP-6187 Fixed: Sequential playback involving 2 Korean voices might cause the voice to spell out text strings rather than pronouncing them.

2025.1.7

  • New Features

    • SP-6166All existing voices have been migrated from the Q22 speech engine to the newer G22 model across all supported platforms. The Q22 language engines and voice data have been removed. Opening your project in the new version of Wwise automatically updates the voice database to the new model but you must manually remove old voice databses (Q22) from the Originals directory.

  • Documentation Improvements

    • SP-5298Improved and added details to the speechEngine documentation. Added additional information on handling the audio markers callback.

  • Miscellaneous Changes

    • SP-6044PS5 SDK 13 is now supported, with G22 voices for all languages.
