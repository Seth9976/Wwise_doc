Starting/stopping the capture process

When you start capturing information from the sound engine, data is automatically recorded in the Advanced Profiler, Capture Log, and Performance Monitor. You can review the information as it is being captured, or you can stop the capture to investigate issues as they arise. If you plan to review the information as it is being captured, you can use the Show Live Data option to keep the Capture Log and Performance Monitor in sync with the capture time.

Note

Each time you start a capture process, Wwise clears the Capture Log. For details on saving the contents of the Capture Log, refer to Saving the Capture Log.

To start the capture process:

  1. Do one of the following:

    • From the menu bar, click Layouts > Profiler.

    • Press F6.

    The Profiler layout is displayed.

  2. If you want to monitor the performance as the data is being captured, click the Show Live Data button, , on the toolbar.

    When this option is selected, the following occurs:

    • The Capture Log will scroll automatically as the data is being captured.

    • The Performance Monitor time cursor will follow the game time cursor.

  3. On the toolbar, click the Start Capture button, , to begin capturing data in the Capture Log.

  4. To stop capturing, click Stop Capture.
