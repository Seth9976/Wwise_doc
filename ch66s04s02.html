Contents Editor: Impacter

The Contents Editor gives you quick access to some of the most common properties associated with the Impacter plug-in. You must click the Impacter source to display the associated column headings.

For a general overview of the Impacter plug-in, refer to Impacter.

Interface Element

Description

Name

The name of the Impacter plug-in.

Use

Determines which source within the sound object is:

  • Played during playback.

  • Included in SoundBanks.

Note

The Use option is only visible when you have more than one source within the sound object.

Notes

Any additional information about the Impacter plug-in.

Mass

Uniquely stretches or compresses the length and pitch of the sound to change its feeling of size or weight. RTPC updates are only applied at the start of an impact.

Default value: 1.0

Range: 0.01 to 2.0

Velocity

Affects the intensity of the impact, changing the timbre and amplitude of the sound. RTPC updates are only applied at the start of an impact.

Default value: 1.0

Range: 0.05 to 1.0

Position

Shapes the timbre of the sound as if the impact was occurring at different positions on a surface. RTPC updates are only applied at the start of an impact.

Default value: 0.0

Range: 0.0 to 1.0

Roughness

Adds a metallic inharmonicity to the sound. RTPC updates are only applied at the start of an impact.

Default value: 0

Range: 0.0 to 1.0

Adds a new source to the sound object.
