Source Editor: Impacter

The Source Editor displays all the properties associated with the Impacter plug-in.

For a general overview of the Impacter plug-in, refer to Impacter.

Interface Element

Description

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Name

The user-defined name given to this instance of the Impacter plug-in.

Source Plugin

The type of source plug-in.

Notes

Any additional information about the Impacter plug-in.

Sets the display of the Source Editor's selected tabs. By default, there is one panel displaying only one selected tab. You can, however, click a splitter button to split the panel into two, either side by side or one on top of the other, for two different tabs. The currently selected option is highlighted with a background color.

Note

You cannot display the same tab in both panels. If you select the tab that is currently displayed in the other panel, then the other panel will automatically display another tab.

Effect Settings

File Input

Audio Source

The name of the audio source file.

Impact Selection

Selects which impact component to add to the SoundBanks to be randomly selected for synthesis.

If there is more than one impact, the random selection always avoids repeating the previously played impact.

The impact is the leftover audio from analysis that drives the synthesis. It can be cross-synthesized with bodies from other files.

Soloing an impact exclusively selects it in the authoring tool for auditioning purposes. The solo setting is not applied to the SoundBanks.

Maximum 64 files.

Body Selection

Selects which body components to add to the SoundBanks to be randomly selected for synthesis.

If there is only one impact available, the random selection only avoids repeating the previously played body.

The body is the set of synthesis parameters extracted from audio analysis. It can be cross-synthesized with impacts from other files.

Soloing a body exclusively selects it in the authoring tool for auditioning purposes. The solo setting is not applied to the SoundBanks.

Maximum 64 files.

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Allows you to browse for additional files.

Deletes all files from the plug-in.

Impact Params

Mass

Uniquely stretches or compresses the length and pitch of the sound to change its feeling of size or weight. RTPC updates are only applied at the start of an impact.

Default value: 1.0

Range: 0.01 to 2.0

Mass Random

Adds a positive random offset to the Mass parameter.

Default value: 0.0

Range: 0.0 to 2.0

Velocity

Affects the intensity of the impact, changing the timbre and amplitude of the sound. RTPC updates are only applied at the start of an impact.

Default value: 1.0

Range: 0.05 to 1.0

Velocity Random

Adds a negative random offset to the Velocity parameter.

Default value: 0.0

Range: -1.0 to 0

Position

Shapes the timbre of the sound as if the impact was occurring at different positions on a surface. RTPC updates are only applied at the start of an impact.

Default value: 0.0

Range: 0.0 to 1.0

Position Random

Adds a positive random offset to the Impact Position parameter.

Default value: 0.0

Range: 0.0 to 1.0

Roughness

Adds a metallic inharmonicity to the sound. RTPC updates are only applied at the start of an impact.

Default value: 0

Range: 0.0 to 1.0

Roughness Random

Adds a positive random offset to the Roughness parameter.

Default value: 0.0

Range: 0.0 to 1.0

Min Duration

Sets a minimum duration before looping or ending a play event. For a minimum duration longer than the impact, silence is appended. RTPC updates are only applied at the start of an impact.

Default value: 0.0

Range: 0.0 to 10.0

Output Level

Level applied to the final signal. RTPC updates are only applied at the start of an impact.

Default value: 0

Range: -96 to 24

Units: dB
