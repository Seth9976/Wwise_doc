SoundBank Editor - Add Tab

The Add tab of the SoundBank Editor is where you build the contents of your SoundBank. You can drag different project elements, including Events, sound objects, and Work Units, from the Project Explorer or Event Viewer to add them to your SoundBank. When you add parent project elements, all corresponding child objects will be added to the SoundBank automatically. For each project element, you must also decide whether you want to include all corresponding events, object structures, and media within the SoundBank.

Wwise uses the following colors to highlight different issues with the project elements that have been added to a SoundBank:

  • Red - The project element is missing from the project.

  • Yellow - The project element has been unloaded from the project.

Interface Element

Description

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Right-click the table header to open the Configure Columns dialog where you can specify which columns to display and their order. See Using tables.

Name

The name of the SoundBank.

Notes

Any additional information about the SoundBank.

Hierarchy Inclusion

The point within the hierarchy at which objects are included in the SoundBank. All objects below this point are automatically included in the SoundBank.

Project elements can be added to a SoundBank manually or by importing a SoundBank definition file. To help you distinguish between the two, an asterisk (*) is added to the name of project elements that have been added manually.

Events

Determines whether the Events associated with the particular project element will be included in the SoundBank.

Structures

Determines whether the sound, music, and motion structures associated with the particular project element will be included in the SoundBank.

Media

Determines whether the media associated with the particular project element will be included in the SoundBank.

Removes the selected project element and corresponding events, structures, and media files from the SoundBank.
