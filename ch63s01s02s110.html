Maximum number of hardware-accelerated voices reached. Voice will not play.

There are too many voices using an hardware-accelerated audio format playing concurrently.

Possible causes:

  • You are using the same conversion settings as another platform that decodes the same audio format in software. Some audio formats may automatically elect to use hardware acceleration on certain platforms.

  • You are using a large number of multichannel, hardware-accelerated sounds. On some platforms, the number of channels counts against the limit, which means you can decode more mono/stereo sources concurrently than 5.1 or 7.1 sources.

  • Virtual voice threshold is set too low.

Recommended resolution steps:

  • Use a different conversion setting for some of the sounds. Consult for advice on which conversion setting is best for different platforms and use cases.

  • Limit the number of concurrent sounds using a hardware-accelerated audio format.

  • Use mono or stereo sources for sounds that are hardware-accelerated.

  • Raise the virtual voice threshold.
