Non-compliant device memory detected. Device memory is required for hardware acceleration.

The Memory Manager used by the sound engine allocated non-compliant device memory. Device memory is platform-specific virtual memory segments with special attributes; these attributes are required for certain hardware-accelerated decoding operations. This is usually the case when using hardware audio decoders.

Probable causes:

  • Your game uses the default implementation of AK::MemoryMgr but has provided incorrect initialization settings for device memory allocation callbacks.

  • Your game uses a custom implementation of AK::MemoryMgr that incorrectly implements the device memory allocation functions.

Recommended resolution steps:

  • Review the Memory Manager requirements in the SDK documentation in how to correctly override the default Memory Manager and verify that your game is fully compliant.
