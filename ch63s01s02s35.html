Bus <name> not found in Init bank.

This error can occur in multiple functions called from the game. The specific function is reported. This error means this Bus ID doesn't exist at the time of the call.

Probable causes:

  • There is a typo in the Bus name.

  • The Init SoundBank was not loaded, or had an error during load.

  • The Init SoundBank was not regenerated and redeployed after a change to the bus structure in the Wwise project.

Recommended resolution steps:

  • Compare the spelling of the name of the bus with those found in the Init.txt file or in the Wwise Project.

  • Regenerate and redeploy the Init.bnk file in your game build.
