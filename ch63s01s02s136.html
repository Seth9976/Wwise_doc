Not enough memory to start sound.

There wasn't enough memory available to allocate all buffers and structures needed to play a sound. The sound will not play. If the overall memory budget is exceeded, this error will occur. This message should be preceded in the Capture Log by Memory allocation failed. Refer to Memory allocation failed. for recommended resolution steps.
