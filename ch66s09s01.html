Source Editor: Sine

The Source Editor displays all the properties associated with the Sine plug-in. You can create different audio tones and effects by modifying the various property values.

Tip

To avoid clicks that may occur when transitioning between pure tones created by the Sine and other one-shot, unlooped sounds, use a fade-in and fade-out of 5 ms.

Interface Element

Description

Name

The user-defined name given to this instance of the Sine plug-in.

Source Plugin

The type of source plug-in.

Notes

Any additional information about the Sine plug-in.

Sine Parameters

Frequency

The frequency of the tone.

Default value: 440.0

Range: 20 to 20000

Units: Hz

Gain

The level or amplitude of the tone’s output.

Default value: -12

Range: -96.3 to 0

Units: dB

Duration

Duration of the tone where the amplitude is constant over time.

Default value: 1.0

Range: 0.001 to 3600

Units: s

(Link/Unlink)

Displays whether the Channel selection is applied across all platforms.

Channel

A list with two options to specify the channel to which the sound is output: Mono and LFE.

Default value: Mono
