Insufficient Space to Load Bank.

There isn't enough memory available to load all objects contained in the SoundBank. Note that this error is related to the Events and Structure part of a SoundBank, not the Media part.

Recommended resolution steps:

  • Split the SoundBank into smaller SoundBanks, loaded dynamically at appropriate times.

  • Unload unused SoundBanks.

  • Increase the total amount of memory available by increasing AK::MemoryArena::AkMemoryArenaSettings::uMemReservedLimit before calling AK::MemoryMgr::Init.
