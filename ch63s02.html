Performance Monitor

The Performance Monitor displays information about the performance, in terms of CPU, memory, and streaming for each activity performed by the sound engine. The information is displayed graphically in real time as it is captured from the sound engine. To the right of the graph view are a series of key performance counters that display the actual performance at each point in the graph. You can review the performance of each counter at specific points in the graph by dragging the Performance Monitor time cursor. You can also press Alt+Comma (,) to move one audio frame to the left and Alt+Period (.) to move one audio frame to the right.

You can add, remove, or change the counters that are displayed in the Performance Monitor in the Performance Monitor Settings dialog. To open the Performance Monitor Settings dialog, click the View Settings icon in the title bar of the Performance Monitor.

Interface Element

Description

Click the View Settings icon in the upper right corner of the view to prompt the Performance Monitor Settings , where you can specify what is displayed in the Graph View and Performance Data list.

Opens a search field where standard alphanumeric entries filter out unmatching elements from the view. Learn more in Using tables.

Click the Close icon to the left of the search icon to close the search field and remove the filter.

Note

The searches do not include elements in collapsed nodes of the List View, Query Editor, MIDI Keymap Editor, and Reference View.

The search is not included in the Settings for Source objects.

Graph View

A series of performance graphs that are updated in real time during the capture process. They display counters , which are specified in the Performance Monitor Settings ' Show in Graph column.

Note

The performance graphs will remain in the graph view until a new capture process is started.

Zooms in towards the center of the Performance Monitor graph view.

Resets the graph view to the default zoom factor ratio of 1:1.

Zooms out from the center of the Performance Monitor graph view.

Performance Data list

A list of performance counters with corresponding values updated in real time during the capture process and displayed according to the selected moment in the graph view's timeline.

The listed counters are specified in the Performance Monitor Settings ' Show in List column.
