Recorder: Cannot create output file.

Pertains to the Recorder plug-in. The plug-in failed to create the output file.

Recommended resolution steps:

  • Verify the output file path provided in the Wwise project.

  • Ensure that the output path has write permissions.

  • Refer to Recorder for more details regarding plug-in configuration.
