Remote Connections dialog

Use the Remote Connections dialog to connect and interact with remote game consoles via a local area network. The Remote Connection dialog displays your current connections status and contains the following two tabs:

  • Available Tab: Displays a list of the remote consoles that were automatically discovered on the network.

  • History Tab: Use to manually create and manage a connection list of game consoles on the network that you have connected to in the past.

Note

When remote connected, the Source Editor's Play Cursor will only work on a Vorbis encoded source if the latter is converted with a seek table.
